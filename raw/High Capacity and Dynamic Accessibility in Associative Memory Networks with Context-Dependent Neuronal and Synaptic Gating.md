## Abstract

Biological memory is known to be flexible—memory formation and recall depend on factors such as the behavioral context of the organism. However, this property is often ignored in associative memory models, leaving it unclear how memories can be organized and recalled when subject to contextual control. Because of the lack of a rigorous analytical framework, it is also unknown how contextual control affects memory stability, storage capacity, and information content. Here, we bring the dynamic nature of memory to the fore by introducing a novel model of associative memory, which we refer to as the context-modular memory network. In our model, stored memory patterns are associated to one of several background network states, or contexts. Memories are accessible when their corresponding context is active, and are otherwise inaccessible. Context modulates the effective network connectivity by imposing a specific configuration of neuronal and synaptic gating—gated neurons (synapses) have their activity (weights) momentarily silenced, thereby reducing interference from memories belonging to other contexts. Memory patterns are randomly and independently chosen, while neuronal and synaptic gates may be selected randomly or optimized through a process of contextual synaptic refinement. Through analytic and numerical results, we show that context-modular memory networks can exhibit both improved memory capacity and differential control of memory stability with random gating (especially for neuronal gating). For contextual synaptic refinement, we devise a method in which synapses are gated off for a given context if they destabilize the memory patterns in that context, drastically improving memory capacity and enabling even more precise control over memory stability. Notably, synaptic refinement allows for patterns to be accessible in multiple contexts, stabilizing memory patterns even for weight matrices that alone do not contain any information about the memory patterns, such as Gaussian random matrices. Overall, our model integrates recent ideas about context-dependent memory organization with classic associative memory models and proposes a rigorous theory which can act as a framework for future work. Furthermore, our work carries important implications for the understanding of biological memory storage and recall in the brain, such as highlighting an intriguing trade-off between memory capacity and accessibility.

## Popular Summary

Memory recall is enigmatic. Sometimes we cannot remember mundane information despite it being on the tip of the tongue, and at other times a chance event may awaken a distant memory from childhood. While experimental neuroscience research has established this dynamic nature of memory as a crucial property, classic associative memory models often ignore it. In this work, we bridge this gap by proposing a new associative memory model that brings dynamic memory accessibility to the fore.

The model features several discrete background states, or contexts, controlling which memories are available for recall and which ones remain hidden. Each context can “gate” off and on particular neurons and synapses, thereby changing the effective network connectivity using a mechanism consistent with what is known about biological neural networks. We explore several variants of this control, including random and optimized. Overall, we show that such a scheme enhances the total memory capacity of the memory network, while also enabling control over which memories can be recalled and which ones remain hidden at each moment.

The end result is a new framework for the study of associative memory that is still amenable to analysis while being much closer to how biological memory works. We believe that this novel theoretical perspective on memory will have important implications for neuroscience, physics, and artificial intelligence.

## Article Text

### I. INTRODUCTION

Discrete attractor networks offer a simple yet powerful model for autoassociative memory, enabling the retrieval of stored, content-addressable memory patterns from noisy external cues . The biological and computational relevance of such associative memory models depends on the statistics of the memory patterns and the choice of connectivity architecture , among other factors. While classic models typically feature uncorrelated patterns (i.e., half of the network units are active for each pattern, chosen uniformly at random) and all-to-all connectivity , many other variants have been proposed and explored . For example, sparse , hierarchical , and structured  patterns have been shown to confer computational advantages to storage capacity and recall, and are more biologically plausible . Furthermore, networks with sparse  or modular  connectivity exhibit improved storage ability and their structure may resemble brain circuits more closely .

A common feature of much theoretical work on associative memory is the focus on memory capacity—the maximum number of stable patterns that a network is able to store . While capacity is certainly a crucial quantity when considering the function of associative memory models, the psychology and neuroscience literature has long highlighted the dynamic and context-dependent nature of memory as an equally significant property . As early as the 1960s, for example, it was observed in free-recall experiments, in which subjects were asked to memorize and later recall lists of words, that sequential cued recall of lists by contextual or conceptual category was often more successful than unconstrained recall of all lists simultaneously . This and related work has led to the proposal that memory *accessibility*—the relative ease by which a particular memory can be retrieved—is not constant, but can vary over time depending upon additional cues and background context . This concept provides an explanation for some memory recall failures, such as the “tip of the tongue” phenomenon in which a memory or a piece of information is momentarily unable to be recalled, suggesting temporary inaccessibility .

Though sometimes inconvenient (such as in the above example), this dynamic nature of memory has the potential to enable powerful flexibility over when and how memories are accessed, which biological memory systems may employ to influence behavior . Given the vast number of memories that humans and other animals may acquire over a lifetime, the ability to dynamically search for and recall specific subsets is likely just as important as the ability to store them in the first place . Furthermore, similar or identical cues often require very different behavioral responses in different contexts—e.g., encountering a dangerous animal at the zoo versus in the wild . Studies of contextual fear conditioning confirm that such fear memories are often linked to particular contexts , and that overgeneralization across contexts may explain pathological memory conditions such as post-traumatic stress disorder . In sum, neuroscience and psychology have reached the consensus that the ability to dynamically store  and recall  memories depending upon the behavioral context is a crucial aspect of memory’s functional role. However, despite several algorithmic models of context-dependent memory , classic network models of associative memory typically feature static and uniform memory stability (i.e., all memories can be recalled at all times with similar accessibility), and thus not only fail in capturing this crucial aspect of biological memory, but also miss out on possible computational advantages of such a flexible approach.

The experimental evidence for dynamic memory accessibility , as well as the potential computational advantages of such flexibility , raises fundamental questions about the circuit-level organization of memory and its theoretical study. First, how can a memory system reliably store a large number of memory patterns while also flexibly regulating their accessibility? Obtaining an answer to this question is crucial for a complete understanding of biological memory and its role in complex behavior. Doing so requires specific hypotheses for the mechanistic basis of dynamic memory stability control in neural circuits. Such mechanisms should enable a network to modulate the stability of particular attractor states on demand in order to read out memories when necessary, but avoid recall when undesired. One promising theory, backed up by a host of experimental evidence, suggests that memories may be bound to specific contexts, requiring reactivation of the encoding context (i.e., reinstatement) for successful recall . Mechanistically, top-down contextual signals from distinct brain areas may act as control signals for memory storage and recall . Previous models have shown that contextual input or gating is a viable option for flexible information routing , as well as continual learning in artificial neural networks . However, it is unclear how such schemes may be included, and rigorously studied, in associative memory models.

This leads to a second important question: How might such a dynamic memory system be studied analytically? Without a theoretical framework and analytical insights, it will be virtually impossible to unveil possible mechanistic, biological implementations of such dynamic accessibility, to understand what the strengths and weaknesses of flexible memory implementations are, and to eventually elaborate predictions for future experiments. Previous contextual gating models have primarily relied upon numerical simulations, making broad computational claims difficult. The large body of theoretical analysis tools based on associative memory models  provides an ideal starting point for the study of contextual control of memory recall, and may serve as the basis to explore whether rigorous theory of flexible memory might have implications for other domains, such as sensory processing and information routing. However, up to this point, it remains unclear how previous mathematical analysis of associative memory may be adapted to such a setting.

Here, we aim to answer the fundamental questions mentioned above by introducing a new model of associative memory, which we refer to as the *context-modular memory network*. In the model, retrieval dynamics are controlled by an externally imposed background state, which we refer to as *context*. This control is exerted onto the network by silencing neurons and synapses, termed *neuronal gating* and *synaptic gating*, respectively. These gating schemes, which are defined in Sec. , modify the effective network connectivity, and thereby also the accessibility of memories, i.e., their stability. We show in Sec.  that Hebbian learning in concert with random gating can considerably improve memory capacity when compared to the classic Hopfield model, especially for random neuronal gating (e.g., up to 7 times more for a network with 100 contexts). We then show that both random neuronal and synaptic gating can effectively stabilize memories in the active context and destabilize memories in the inactive context, thereby modulating memory accessibility. Interestingly, these results may indicate that low levels of neuronal activity found in the brain (and their allocation to particular memories ) might not only serve to increase memory capacity, but also to flexibly control memory recall. We then introduce the concept of *refinement* in Sec. , in which synaptic gating (but not neuronal gating) can be optimized to improve memory stability for specific groups of previously learned patterns. This scheme results in a drastic increase in memory capacity (e.g., up to 40 times more for a network with 100 contexts), plus an even more powerful ability to modulate memory accessibility. Finally, we show that overlapping sets of memories across different contexts can be stabilized through synaptic refinement, demonstrating how contexts may share subsets of common memories. The precise control needed to implement synaptic refinement suggests novel hypotheses for ongoing plasticity following initial memory storage, which can be explored experimentally. Overall, this work incorporates novel ideas about the dynamic and behaviorally relevant use of memory into classic associative memory models, and may serve as a novel framework for the study of flexible associative memory.

### II. MODEL

#### A. Model description

We introduce a context-dependent autoassociative memory network, the context-modular memory network, whose primary functionality is to have dynamic memory accessibility—that is, depending on a background context signal, the model stabilizes a subset of memory patterns while keeping others hidden. This is achieved by temporarily modifying the network such that there is a different effective connectivity matrix for each contextual state \[Fig. \]. To implement such a model, we modify a standard, fully connected Hopfield network with 𝑁 neurons  to include 𝑠 contextual states, i.e., contexts, which control two types of gating. First, in neuronal (nrn) gating, each context (cxt) has a corresponding set of allocated neurons 𝑁cxt, while the other 𝑁 −𝑁cxt neurons are “gated off” (defined more precisely below). We use the parameter 𝑎 \=𝑁cxt/𝑁 to denote the proportion of allocated neurons per context. Second, in synaptic (syn) gating, each contextual state defines a particular set of allocated connections (synapses) per allocated neuron 𝐾, with the other 𝑁cxt −𝐾 connections being gated off. We use the parameter 𝑐 \=𝐾/𝑁cxt to denote the connection density.

#### B. Model interpretation

The gating schemes defined above can be interpreted as mechanisms that impose a distinct effective connectivity matrix for each contextual state. The network dynamics can then be viewed as following a context-dependent energy function \[Fig. \],

with context 𝑘 being active. Neuronal gating temporarily removes particular rows and columns from the connectivity matrix \[Fig. , left\], whereas synaptic gating temporarily removes individual entries \[Fig. , middle\]. The combination of the two produces a smaller and sparser connectivity matrix \[Fig. , right\], with a potentially large number of synaptic connections remaining hidden, to be used in other contexts. This enables the fully connected network to exhibit an effectively modular structure, in which the active context 𝑘 imposes an *inhibitory mask*, given by 𝑎𝑘𝑖⁢𝑎𝑘𝑗⁢𝑐𝑘𝑖⁢𝑗, onto the connectivity matrix. It will be useful to analyze the model in terms of the proportion of gated neurons, i.e., neuronal gating ratio, 1 −𝑎, and proportion of gated synapses, i.e., synaptic gating ratio, 1 −𝑐. The neuronal gating ratio indicates the proportion of neurons that are gated off while a given context is active. The synaptic gating ratio, usually referred to as dilution , indicates the proportion of synapses that are gated off during recall of memories in the active context.

A crucial feature of the model is the assumption that only a subset of memories needs to be stable in each context. As the number of contexts 𝑠 increases, the proportion of accessible memories decreases as 1/𝑠 \[Fig. \], leading to a clear trade-off between the storage capacity gains (which increases with 𝑠) and memory accessibility across all patterns at any given time. This also means that the proportion of inaccessible memories grows with the number of contexts, making it just as important to ensure that the hidden memories are unstable as it is to ensure that the accessible memories are stable.

#### C. Memory capacity definitions

We define parameters related to pattern storage to study the maximum number of memories that can be successfully retrieved. The network *load* refers to the ratio of stored patterns to network size. Here, we have two notions of load. First, the *single-context load* is defined as

𝛼cxt\=𝑝/𝑁cxt,

(3)

which is the number of patterns stored in each individual context divided by the number of allocated neurons per context. For simplicity, we impose that all contexts store the same number of patterns and have the same number of allocated neurons, and so the single-context load is identical across all contexts. Second, the *total load* is

𝛼\=𝑃/𝑁\=𝑠⁢𝑝/𝑁\=𝛼cxt⁢𝑠⁢𝑎,

(4)

which is the total number of patterns stored in the network across all contexts divided by the network size. The maximum load for which all patterns are stable attractors is known as the memory capacity. We refer to the single-context and total capacities as 𝛼\*cxt and 𝛼\*, respectively . Because of analytical tractability, memory capacity is probed conditioned on an externally imposed context (see Sec. ).

### V. DISCUSSION

The growing consensus perspective in neuroscience and psychology views biological memory not as a static storage device, but rather as a dynamic and flexible system . Such a system adapts to current behavioral states, i.e., contexts, and is thus able to modify the understanding of past experiences when needed . Here, we revisited classic theoretical work on autoassociative memory in light of this perspective. Specifically, we proposed a novel context-modular memory network model in which the memory patterns and network connectivity are modified through neuronal and synaptic gating by a background contextual state. We found that such a scheme not only results in increased memory capacity, but is also effective at modulating the stability of stored patterns, thereby selecting a subset of memories for recall and reducing interference from others. Improved memory capacity is thus not the main feature of the context-modular memory network, but a side effect of increased control and flexibility which is traded off with the proportion of accessible memories at any given time.

We explored two contextual gating schemes: neuronal and synaptic. To implement these schemes, an inhibitory mask is imposed onto the recurrent connectivity to modulate the energy function of the network dynamics. Neuronal gating acts to remove particular neurons from the active context, thereby shrinking the effective network size and removing interference from masked neurons. The potential biological mechanisms for neuronal gating are numerous, including perisomatic inhibition  and modulation of intrinsic excitability . Synaptic gating features the masking of individual synaptic elements, thereby making the network connectivity more sparse (diluted) in a particular context, reducing interference between accessible and inaccessible memory patterns. Though the precise and complex nature of synaptic gating makes individual synaptic control less plausible, there is some evidence of inhibitory connectivity at the level of individual dendritic spines, which could be interpreted as synapse-level control . However, there is considerably more evidence for control of synaptic inputs at the level of dendritic branches , supported by the presence of specific dendrite-targeting inhibitory interneurons in both cortex and hippocampus , and findings such as dendritic clustering , localized dendritic inhibition , and localized neuromodulation . These mechanisms could allow for the electrical isolation of particular areas of the dendrites , and clustering and control of specific input groups , thereby implementing a dendritic-branch-specific gating. One area for future work would be to explore this by relaxing the precision of synaptic gating, for which the analytical techniques described here may already be useful.

We proposed two methods for setting contextual states in our model: random allocation and refinement. In random gating, contextual states are imposed at the time of learning, which preallocates neurons and synapses to store memory patterns in a given context, and does not affect gated components. For random gating to be effective, contextual states are also reinstated during recall—such contextual reinstatement is hypothesized to occur in the brain . We observed that neuronal gating was more effective than synaptic gating for random allocation, resulting in high memory capacity (Fig. ) and low stability of inaccessible memories (Fig. ). Such a finding is intriguing in light of recent evidence for allocation of particular neurons (through excitability changes) to memories in the amygdala and hippocampal system . Specifically, the activity levels observed in such studies ( ∼20%–30%) correspond to optimal neuronal gating levels in our model for tens to hundreds of contexts (Fig. ). Importantly, silencing of the allocated neurons—i.e., neurons with elevated excitability during memory formation—has been shown to decrease memory expression , in line with random neuronal gating during both learning and recall. Our model may be used to propose additional, context-dependent manipulations that may artificially bias recall toward or away from particular memories.

The second contextual gating method that we proposed was refinement, in which gating states are optimized to strengthen accessible memories after learning (i.e., contextual states are imposed only at retrieval). This scenario was proposed specifically with synaptic control in mind, as synapses were gated off in a particular context if they had a net harmful effect on recall. We found that synaptic refinement resulted in even larger gains in memory capacity (Fig. ) and better control of memory accessibility (Fig. ), with connectivity levels approaching 50% (Fig. ). Such a scheme fits well with the idea that memories are constantly being updated , with each recall event possibly leading to changes . Furthermore, recent work has also suggested that memory expression can be masked and controlled through inhibition . Additionally, the finding that synaptic refinement can stabilize shared groups of memory patterns (even using a random weight matrix) suggests that the precision of synaptic learning may not be crucial to long-term maintenance of memories (see Refs.  for related studies). Lastly, we note that since refinement was defined at the level of synapses rather than neurons, we did not consider the refinement of neuronal gating here. However, while *random* neuronal gating proved to be surprisingly effective, it was still the case that many synapses remained unused when capacity was maximized \[Fig. , inset\]. This suggests that a more algorithmic allocation process (e.g., one that equalizes the number of contexts that a neuron participates in) may be possible, which would be interesting to explore in future work.

Memory capacity, stability, and information content results were obtained using conditional measures—i.e., we assumed an externally controlled context, with each measure conditioned upon this fixed external state. This choice was made due to reasons of analytical tractability, and the desire to model a flexible memory system more related to biological memory . Furthermore, the questions addressed here did not depend upon an explicitly specified context representation, as the focus was rather on the consequences of contextual control for an associative memory network. However, our conditional measures make comparisons with previous capacity estimates difficult, as performance metrics usually require stabilizing all memories at all times . We therefore stress that our aim was not to compare the capacity with previous networks, but rather to propose a framework within which to study capacity while emphasizing flexibility and accessibility. In future work, an explicit implementation of a context-encoding network may potentially lead to interesting biological comparisons, including dynamic competition between memories , as well as descriptions of pathological memory states like in amnesia . Overall, there may be a trade-off between capacity and flexibility, and it may be in the interest of the brain to favor dynamic accessibility over total amount of memories stored. Theoretically, it will be necessary to develop richer measures than capacity to determine the use and function of memory that includes flexibility, retrieval dynamics, and evolving memory patterns .

One simple way of considering the complexity of such contextual representations in the capacity estimations is to consider a Hopfield-like (context-encoding) network storing contextual states as memory patterns. For neuronal gating, our model would require the size of such a network to scale with the number of contexts 𝑠, which should be relatively small compared to the memory network size, and thus result in qualitatively similar results. For synaptic gating, however, our model would require fundamentally different dynamics in which synaptic weights are controlled by activities of other neurons . Assuming this is possible, and constraining each context-encoding neuron to have on the order of 𝑁 synapses, the network should scale with 𝑠⁢𝑁, leading to a context network that is potentially orders of magnitude larger than memory network, and reducing the memory capacity below the Hopfield limit with only a few contexts. Such intuitions argue against full and precise synaptic control. However, relaxations of such precise control, especially for a smaller number of contexts, may be possible, and our model serves as a proof of concept of such a situation.

Context-modular memory networks utilize several principles in common with previous associative memory models, including sparsity, modularity, and hierarchical organization . Modularity can be used to mimic the architecture of the cortex—dense local with sparse long-ranged connectivity —and several other models have explored a hierarchical organization of memories , or nontreelike organization of semantic memories . Furthermore, other models have incorporated selective connectivity to enhance capacity . Our model, in contrast, is unique in considering context as the main determinant of memory groupings (but see Ref. ), as well as modeling context as an external signal imposed on the memory network (in contrast to previous hierarchical models). However, more generally, the modularity of the model plays the same important role as in these previous models, namely to reduce crosstalk and interference between memories. By exploring the amount of overlapping connectivity \[shared contexts per synapse, Figs.  and \], we found that the effectiveness of random gating is in part due to its ability to isolate the weights used for different contexts. Despite this, we still found that for most parameter settings, the majority of the weights were used in more than one context, and that this is crucial to the capacity gains seen in the model.

Extensions to associative memory networks that improve memory capacity, such as low activity patterns , alternative learning rules , or complex synapses , may be incorporated into the context-modular memory network architecture for additional benefits. We note that this also may include connections to recent advances in dense associative memory with exponential capacity , and heteroassociative memory architectures . Furthermore, we note that while many of these classic models use very simple binary units for the neurons, there have been many successful extensions of this work to more realistic single-neuron dynamics . While we limited ourselves here to the simple binary case, the neuroscientific implications of the work may be better explored in more realistic models. We also assume symmetric connectivity throughout, but a more realistic scenario would be asymmetric connectivity, for which other analyses apply (see Refs.  for details on asymmetric matrices).

Finally, other models have considered contextual control of learning and memory , as well as similar gating mechanisms applied in other situations (e.g., cortical information routing , deep learning , inhibitory learning , and motor learning ), highlighting the importance of contextual flexibility for cognition. Our work may serve as a theoretical framework with which to begin to study contextual gating in such additional scenarios.

In conclusion, the integration of associative memory models with retrieval processes and decision making is a promising area of future research . Organizing memories around context leads to control over memory access, which the brain may harness. We propose the contextual gating framework introduced here as a promising model for flexible memory access which may lead to new insights into biological and artificial memory.
