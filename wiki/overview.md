# Overview — Brain-Inspired Models for Abstract Reasoning

---

## The Central Thesis

Current deep learning fails at out-of-distribution systematic generalization. Biological brains do not. The hypothesis: the brain treats all reasoning as **latent graph discovery** — infer the hidden relational structure of a domain from observations, then navigate it. The key architectural principle enabling this is **factorization of structural knowledge from sensory content**, extended by a BG (Basal Ganglia)/PFC control layer that gates, modulates, and hierarchically structures the inference process. The hardware is spiking cortical columns running a canonical microcircuit; the learning algorithm is contrastive Hebbian at energy equilibrium. **Memory schemas = slow W:** mPFC schemas are organized overlapping association networks — the brain's latent graphs; schema formation is biological latent-graph discovery; schema assimilation/accommodation is controlled consolidation without catastrophic interference. **Planning as inference:** given a world model embedded in recurrent weights, the optimal action sequence is inferred in parallel via attractor dynamics, not sequential rollout — the Spacetime Attractor (STA) ([[wiki/entities/spacetime-attractor.md]]) implements this in PFC subspace circuits, avoiding rollout error accumulation. **The contextual layer:** above the BG/PFC control layer, the DMN (Default Mode Network) maintains a continuously updated "frame of thought" — integrating episodic memory, semantics, and self-reference — that serves as the slowly changing contextual scaffold within which fast WM/attention operates ([[wiki/entities/default-mode-network.md]]). Formal grounding: a system **knows** a domain (Choi 2026) when its intelligence density $\mathcal{I} = \log_2 N / C \to \infty$ as the domain scales — a criterion recurrent/iterative architectures can satisfy but fixed-depth feedforward networks cannot.

**Reasoning is a coupled two-system loop.** The mechanistic stack resolves into a metric **Navigator** (System 1: MEC/HC path-integration — *where am I / what will I see*) and a declarative **Strategist** (System 2: PFC/mPFC — *where do I want to go / how*), joined by a continuous↔discrete codec and driven by two prediction errors: a transition error $\varepsilon_x$ that trains the operator vocabulary ($W$, $V(a)$) and a value error $\varepsilon_r$/dopamine that trains the goal. Dopamine's dual role (RPE + novelty) is why a *goal-free* agent can bootstrap a goal — the same error that builds the world model drives the exploration that discovers what to want. The loop is trained by a three-phase curriculum — **explore** (curiosity-driven SSL) → **imitate** (behavioral cloning of human step-traces, which supplies the latent path for free) → **practice** (verifier-gated RL) — and compiles effortful System-2 search into fast System-1 intuition via offline replay ([[wiki/queries/reasoning-as-coupled-navigation-strategizing.md]], [[wiki/queries/proposed-reasoning-model-block-architectures.md]]). **The two systems are Penn's two:** System 1 is proto-symbolic and shared with animals; System 2 is the human-only *graft* that reinterprets first-order relations as role-governed structure (a Physical Symbol System) — a graft, not a scale-up. Above the mechanistic stack sits a **uniquely-human layer**: innate **core-knowledge** priors (objects/agents/number/geometry — the extreme-prior end of slow-W), a **recursion**/discrete-infinity combinator (the missing productivity engine, ≡ the formal content of the multi-level meta-graph gap), a **cultural** acquisition channel (imitation over emulation — vocabulary is transmitted, not reinvented), and a **neoteric** developmental window (the extended plastic phase that *is* the long outer loop).

## Master Problem Framing: Latent Graph Discovery

> **Latent graph discovery = infer the structure (nodes, edges, topology) of a relational graph from observations, then navigate it — where the graph is never given explicitly.**

Every cognitive task reduces to which *subset* of graph components is hidden. The taxonomy is a set of **independent latent variables** — a task sets a *subset* of hidden bits (a **bit-vector**), not one exclusive type — so benchmarks legitimately span multiple columns. Crossing the canonical benchmark families against the six latent variables (✓ latent · ◐ partial · — given):

| Benchmark family (examples) | Node content | Topology | Edge labels | Vocabulary | Path | Goal node |
|---|---|---|---|---|---|---|
| **Rule induction** — ARC-AGI, IQ tests, analogy | — | — | ✓ | — | — | — |
| **Interactive rule + goal** — ARC-AGI-3 | ◐ | — | ✓ | ✓ | ◐ | ✓ |
| **Path, known vocab** — Navigation; AIME/HMMT; MATH | — | ◐ | — | — | ✓ | — |
| **Path, partial vocab** — OlymMATH-HARD | — | — | — | ◐ | ✓ | — |
| **Node-content-latent QA** — GPQA; algebra, physics, CSP | ✓ | — | — | — | ◐ | — |
| **Edges + path + vocab** — FrontierMath | ◐ | — | ✓ | ✓ | ✓ | — |
| **Topology discovery** — scientific/causal discovery | ◐ | ✓ | ✓ | ✓ | ✓ | ◐ |

Difficulty reads off the row weight — more ✓ = more hidden mass = harder — but this is a **derived difficulty ordering, not a partition**: FrontierMath sets three bits at once (edge-labels + path + vocabulary), which is why a "task type" is a bit-vector and appearing in several columns is expected. The edges-vs-path distinction is *single-hop rule identity* (ARC-AGI: infer one transformation from before/after pairs, vocabulary being learned) vs. *multi-hop composition over an already-known move-set* (navigation, proof search). Full axes and reading notes in [[wiki/concepts/latent-graph-discovery.md]].

Every domain has a **two-level hierarchy**: a slow meta-graph (shared rules across episodes) and a fast instance-graph (task-specific topology). Systems that conflate these levels must relearn rules from scratch for every new problem. The W/M two-timescale split ([[wiki/concepts/two-learning-timescales.md]]) is the direct computational implementation: W = meta-graph, M = instance-graph. **Memory schemas** are the mPFC's biological implementation of slow meta-graph storage: the schema IS the recovered latent graph; schema selection = choosing which latent graph to navigate; schema assimilation/accommodation = controlled meta-graph update. **The Chomsky hierarchy grades meta-graph expressiveness** ([[wiki/concepts/recursion.md]]): a finite-state/Markov transition model is *provably insufficient* for rules-embedded-in-rules (the AₙBₙ center-embedding regime tamarins fail and humans pass); the meta-graph must be phrase-structure (recursively nested), which is the formal content of the multi-level meta-graph gap (Gap #2). Discrete infinity ≅ the natural-number successor function — a model that acquires the integer successor rule has, in miniature, language's generative operator.

**Primary empirical target:** ARC-AGI — latent edge discovery from sparse (before, after) grid pairs. ARC-AGI-2 resists brute-force in verified settings (open Kaggle SOTA 24%, audited harness 54%; frontier labs self-report ≤85% on closed models, unverified). ARC-AGI-3 extends to interactive agents that must also discover the target goal *and* the latent per-game effect of each action, building an adaptable world model from exploration alone — and its OOD private set / power-law scoring make it strongly contamination-resistant, a key reason it is a good contender for validating genuine reasoning. **VSA/HRR isolates the bottleneck:** 94.5% Sort-of-ARC (pre-given DSL) → 3% ARC-AGI-1-Eval (open DSL) = vocabulary co-discovery is the residual gap once path integration and binding are solved (Joffe & Eliasmith 2025).

**Hardness source 5 — spurious edge covariate shift:** training observations contain false edges that work i.i.d. but fail o.o.d. ANLI (Nie et al. 2020) quantifies this cleanly: NLI hypothesis-only baseline 72% IID → 42–51% when the spurious H→label edge is adversarially blocked, forcing traversal of the true P→H entailment edge. Larger LLMs are *more* susceptible (inverse scaling paradox, Yuan 2024) — capability accumulation amplifies false-edge availability. CoT (Chain of Thought) partially bypasses shortcuts by forcing multi-hop traversal rather than single shortcut edges, but cannot fix incorrect edge vocabularies.

**Hardness source 6 — non-stationary topology (new):** the edge set *itself* rewrites within a single episode (rules change, doors open, other agents alter reachability) — the only source that breaks the stationarity assumption the other five share, since it mutates the instance-graph *inside* the fast-M binding window. It is learnable **iff the rewrite process is itself a stationary higher-tier rewrite-graph** — recursing the W/M hierarchy one level up — and is unsolvable *in principle* (information-theoretically, not merely uncomputably) when rewrites are truly random with no compressible generator. **The resolution that preserves W/M** (the *rule-state-as-dimension* lift): augment the node `s' = (base_state, rule_config)` and the graph is stationary again — the exact analogue of folding time into the node (STA/SR). This costs state-space blowup unless the rule-config *factorizes*; it reduces to **two** timescale levels for Baba-class rewrites (stationary rule-change generator → slow W; current active rule-config → continuously-updated fast M, no longer write-once) and requires an irreducible **third** level only for Nomic-class self-amendment (a mutable rule that edits the rule governing rule-changes). Distinct from the controllable-vs-exogenous edge split (a refinement of what "Edges" means — `a_t = ∅` selects a fixed autonomous-drift edge, so topology stays stationary — *not* a hardness source). **Baba Is AI** ([[wiki/entities/baba-is-ai.md]], Cloos et al. 2024) is the empirical counterpart: rules are movable tiles the agent rewrites, pinning all four tractability axes (legible, controllable, bounded, meta-stationary) to their easy pole — yet GPT-4o/Gemini score ~15–20% on composing rewrites (`break→make→goto`), placing the current-model bottleneck far below the in-principle ceiling: models cannot plan over an edge set they themselves edit even when every edit is observable and finite. See [[wiki/queries/dynamic-graph-vs-time-dimension.md]] and [[wiki/queries/rule-changes-as-meta-graph-rules.md]].

## Current Best Understanding

### Six Core Architectural Principles

| # | Principle | Brain substrate | Why needed |
|---|-----------|----------------|------------|
| 1 | **Factorize structural code from sensory code** | g (MEC) vs. x (LEC) | Structure W generalizes; content stays environment-specific. Without factorization, relational generalization lies outside the model's reachable manifold — structurally impossible regardless of training |
| 2 | **Maintain latent states via path integration** | HC / g-update RNN | De-alias observations; compress rule learning to O(relation types) not O(transitions) |
| 3 | **Two-timescale learning** | Slow W (cortex/MEC) + fast Hebbian M (HC) | Slow W builds shared meta-graph; fast M stores episode-specific instance-graph |
| 4 | **Gate and modulate via BG (Basal Ganglia)/neuromodulation** | Basal ganglia + DA/ACh/5-HT/NA | Credit assignment: D1→LTP (Go), D2→LTD (NoGo) from a single phasic Dopamine event; ACh gates storage vs. retrieval |
| 5 | **Hierarchical control** | PFC (Prefrontal Cortex) rostro-caudal gradient (BA-8 → BA-9/46 → BA-10) | Rules about rules: lower levels handle stimulus-response; mid-lateral maintains task-set context; frontopolar manages branching rule chains |
| 6 | **Canonical hardware substrate** | L4→L2/3→L5→L6→L4 microcircuit | Recurrent amplification of weak inputs; horizontal WTA (Winner-Take-All) selects hypotheses; SLN% (proportion of Superficial Layer Neurons) encodes hierarchy; explore (superficial) / exploit (deep) split maps to inference / action |

### Key Unification Results

- **TEM = transformer** — the factorized key/value structure (Q=K=f(g), V=f(x)) follows necessarily from the outer-product memory; Hopfield ↔ attention is grounded in the Boltzmann distribution (softmax IS P ∝ exp(−E/T))
- **Grid cells = SR (Successor Representation) eigenvectors = optimal path-integration bases** — the periodic grid code is optimal for graph matching and multi-step planning via the same mechanism
- **Place cells = SR (Successor Representation) rows = p = f(g, x)** — "anomalous" HC cells (splitter, lap, evidence) are the same model with hidden task dimensions added to the graph
- **FEP grounds TEM's objective** — the two-timescale training objective (slow W, fast M) is the factorized ELBO; all cross-entropy training is Bayesian brain inference
- **EqProp proves local Hebbian is exact backprop** — Theorem 1 (Scellier & Bengio 2017): ΔW ∝ (1/β)[ρ(u^β)ρ(u^β) − ρ(u⁰)ρ(u⁰)] computes ∂J/∂W exactly as β→0; the free phase IS the inference phase; no separate backward circuit required. Resolves the theoretical half of Gap 4.
- **Canonical microcircuit = canonical PC (Predictive Coding) circuit** — Bastos et al. 2012 maps PC's error/prediction split onto L2/3 (error, gamma, feedforward) vs. L5/6 (prediction, beta, feedback); spectral asymmetry is a mathematical consequence of the laminar anatomy
- **D1/D2 opponent plasticity = cellular credit assignment** — a single phasic Dopamine event simultaneously drives PKA→LTP in direct (Go) SPNs and EC→CB1→LTD in indirect (NoGo) SPNs; credit assignment in the brain is structurally opponent, not scalar
- **PVLV implements this biologically** — Dopamine drives slow W update; ACh gates the M write/read switch; NA (Noradrenaline / Norepinephrine) sets exploration breadth; 5-HT sets planning horizon
- **SNN type B > sigmoidal > threshold** — Maass 1997 Theorem: a single type B spiking neuron computes CD_n (coincidence detection) with Ω(n²) fewer sigmoidal units. Synchrony codes are exponentially cheaper than rate codes for binding.
- **Intelligence density formalizes knowing** — Choi 2026: $\mathcal{I}(S) = \log_2 N(S) / C(S)$ unifies Legg-Hutter $\Upsilon$, Chollet's efficiency metric, and Kolmogorov complexity under one criterion; a system *knows* a domain iff $\mathcal{I}(n) \to \infty$ as domain scales; recurrence is the minimal architectural condition; feedforward networks are provably bounded to $\mathcal{I} = \Theta(1)$
- **Mode-2 ≠ System II (Lett 2025)** — LeCun's Mode-2 planning (MPC with learned world model + hard-wired search) is stochastically deterministic: same algorithm, same world model → same plan. True System II requires the *search/planning algorithm itself* to be learned. V-JEPA 2-AC concretizes Mode-2 at scale: CEM in latent space achieves zero-shot pick-and-place (65–80% success) but relies on a fixed CEM algorithm — the planning procedure is not adapted.
- **Representation-space prediction outperforms generative for planning** — V-JEPA 2-AC vs. Cosmos (pixel-space): 65–80% pick-and-place success vs. 0%; 16 sec/action vs. 4 min/action; VL-JEPA (1.6B) outperforms GPT-4o, Claude-3.5, Gemini-2.0 on WorldPrediction-WM (65.7%) without generating a token
- **STA = planning as inference = ring/grid attractor extended to spacetime** — Jensen et al. 2026: the STA embeds the world model (environment adjacency matrix A_{ij}) in inter-subspace connections and infers the optimal T-step trajectory as a fixed point; conveyor belt dynamics execute without replanning; same "consistent states excite, inconsistent inhibit" principle as ring attractors but extended over space × time. Meta-trained RNNs spontaneously discover STA-like representations (recurrent weights r=0.91 with adjacency matrix). See [[wiki/entities/spacetime-attractor.md]].
- **Schemas = latent graphs = mPFC slow W** — memory schemas (Preston & Eichenbaum 2013) are organized overlapping association networks in mPFC; episodes are nodes, shared elements (B connecting A–B and B–C) are edges; schema formation IS biological latent-graph discovery. The vmPFC→MEC→NGF circuit (de Sousa et al. 2026) implements schema-guided co-allocation of new HC memories into prior schema engrams: vmPFC high → NGF active → dCA1 overlap low → accommodation; vmPFC low → NGF quiet → dCA1 overlap high → assimilation. A 7-day gate corresponds to systems-consolidation of the prior schema. See [[wiki/concepts/memory-schemas.md]].
- **PFC hierarchy = parallel multi-level policy search** — Badre et al. 2010: all frontal levels (PMd/1st-order → prePMd/2nd-order → mid-DLPFC/3rd-order → FPC/4th-order) search simultaneously from trial 1; reward-gated depth pruning (prePMd activates for 2nd-order search before any 2nd-order rule is known); step-wise sigmoid learning curves as the empirical signature; directed putamen→cortex→caudate Granger-causality chain; abstract rule discovery requires the full hierarchy even when only 1st-order rules are currently available.
- **HC specificity = spatial/temporal schemas only** — Kumaran & Maguire 2005 factorial fMRI: HC uniquely activates for spatial relational graph traversal but NOT for topology-matched social relational graphs; amodal associative schemas are organized by mPFC via STS/TPJ/temporal pole. The TEM-HC combination is the right substrate for spatial/sequential rule learning; purely abstract logical schemas require a different substrate.
- **Two distinct latent-structure subsystems, not one** — continuous / metric / path-integrable structure recruits the MEC→HC→vmPFC/DMN grid scaffold (a *content-free periodic hash* that turns cross-instance graph-matching from NP-hard search into a phase shift; Constantinescu 2016 finds the hexagonal code strongest in vmPFC, not EC, during abstract 2D "navigation"); arbitrary *discrete/declarative* rule structure recruits the PFC rostro-caudal hierarchy + mPFC schema system (reward-gated RL search, no metric embedding). "Abstraction" is not one thing — navigation is just the special case where the driving input is self-motion velocity. This sharpens Gap #2: the meta-graph is potentially **two structurally different representational formats that must gate each other** (e.g. "apply rule X only in region A of a continuous space"). See [[wiki/queries/mec-abstract-codes-vs-declarative-rules.md]].
- **Single neurons store abstract task rules (Miller 2002)** — PFC single units carry sharp-boundary category tuning (remappable to orthogonal category schemes), abstract rules that generalize to novel stimuli (compression over paired associates), and nonlinear conjunctive tuning — the single-cell origin of mixed selectivity — acquired in 5–15 trials. Grounds the meta-graph's edge labels at the single-cell level and distinguishes two representational formats within one circuit: **sustained activity (fast M) vs. synaptic weights (slow W)**. See [[wiki/papers/pfc-categories-concepts-miller-2002.md]].
- **Memorization vs. generalization has a formal boundary (Nanda 2023)** — the micro/macrofeature criterion: a task is *pure memorization* exactly when no macrofeature (input-general, task-relevant regularity) predicts the output beyond raw entity identity; in that regime distributed/superposed lookup-table storage is provably the *only* option and generalization is impossible — it is not an interpretability failure. This delineates where latent-graph discovery even applies (a degenerate graph — every node its own free parameter, no shared edges — has nothing to discover). Empirically, Pythia 2.8B stores facts via *dense polysemantic superposition* (near-zero cross-fact neuron correlation), not sparse localist engrams — a tension with biological SDR coding, possibly reflecting one-shot-write (biological HC) vs. batch-consolidation (LLM pretraining) constraints rather than a true contradiction. See [[wiki/papers/fact-finding-factual-recall-nanda-2023.md]].
- **Non-stationary topology recurses the W/M hierarchy (Baba/Fluxx/Nomic)** — rule-changing games are natural experiments in *what keeps a mutating edge set learnable*: all stay playable only by pinning four axes (legibility, driver, meta-stationarity, bounded vocabulary) away from randomness. The rule-state-as-dimension lift restores stationarity; two timescale levels suffice for Baba-class self-editing, three for Nomic-class self-amendment; truly random rewrites are unsolvable in principle. See hardness source 6 above and [[wiki/entities/baba-is-ai.md]].

### The Coupled Reasoning Loop and Training Curriculum

Navigation gets three things for free that reasoning must discover: a known action vocabulary, a given goal, and a straight path to it. The resolution is a **coupled loop between a metric Navigator (System 1) and a declarative Strategist (System 2)** whose single tandem prediction splits into two errors that train different modules ([[wiki/queries/reasoning-as-coupled-navigation-strategizing.md]]):

| Latent thing (reasoning ≠ navigation) | Discovered by | Signal |
|---|---|---|
| **Vocabulary** (operator labels) | inverse path integration (Block 3A: infer operator from $\varepsilon_x$) | transition error $\varepsilon_x = x' - \hat{x}'$ → trains $W$, $V(a)$ |
| **Goal** (no external reward yet) | neuromodulatory valence + curiosity bootstrapping | $\varepsilon_r$/DA; DA-novelty makes $\varepsilon_x$-reduction *itself* the reward |
| **Path** (sequence of operators) | replay-generated, verifier-scored refinement | HC/DMN preplay generates chains → PFC value head scores vs. $g_{goal}$ |

**Dopamine does double duty** (RPE for value + novelty for exploration): the same error that builds the world model drives the exploration that discovers the goal — the cleanest statement of why a goal-free agent can bootstrap goals, and a capability absent from all current architectures (Block 3D gap, the ARC-AGI-3 crux). Under diffuse `g` the intrinsic reward is *not* raw novelty (hijacked by the noisy-TV/aleatoric trap) but a blend of **learning progress** ($-d/dt\,E[\varepsilon_x]$, self-organizes a Goldilocks curriculum), **information gain** (posterior-entropy drop over $W$), and **empowerment** ($I(A;S')$, reach controllable states). Training runs a three-phase developmental curriculum: **① explore** (curiosity-driven world-model SSL; learns vocabulary) → **② imitate** (behavioral cloning / process supervision on human step-traces — supplies the latent *path* for free, the single most expensive object; short-circuits search) → **③ practice** (RL/refinement on the agent's own rollouts against a verifier — the only phase that survives off-distribution and can *exceed* the teacher; AlphaGo pattern). Imitation alone provably caps out (covariate shift + self-poisoning), so Phase 3 is required; in the verifier-free ARC-AGI-3 regime the intrinsic LP+empowerment signal is the load-bearing substitute for the missing external verifier. As `M` consolidates into schema and `V(a)`/`g_goal` sharpen, the same problem is solved by near-straight System-1 path integration — **effortful System-2 search compiled into System-1 intuition** via offline replay, for free in this design. The analytic ancestor of the Navigator's generalizing completion is the **Temporal Context Model** ([[wiki/concepts/temporal-context.md]]): its leaky self-normalizing integrator `t_i = ρ_i t_{i-1} + β t_iᴵᴺ` is the scalar-ρ precursor to `g`, already yielding transitive inference as a similarity gradient over co-occurrence statistics — the primitive, fully-analytic form of SR's γT and TEM's learned `W`.

### The Uniquely-Human Layer: What System 2 Must Add

The comparative-cognition literature sharpens the model-building/pattern-recognition split into a **representational specification** for System 2 and names the innate scaffold, the combinatorial engine, the acquisition channel, and the developmental window:

- **Relational reinterpretation = the System-2 spec (Penn, Holyoak & Povinelli 2008)** — every mind builds *functionally* compositional, *featurally* systematic first-order relations (System 1, shared with animals); only humans add a system that reinterprets them as role-governed, role-filler-independent **Physical-Symbol-System** structure. The four PSS properties a reasoner must add: role-filler independence, type/token separation, concatenative composition, classical (structural) systematicity. **Chunking + segmentation is how System 1 *fakes* relational reasoning** — collapse a relation into an analog scalar (entropy/feature-count), evaluate as chunked conditional discriminations — which is the *representational-level root of shortcut reasoning* and the default attractor discriminative objectives converge on (pigeons find 16-item same/different *easier* than 2-item). **"Graft, not scale":** the second system required a distinct binding mechanism (LISA's gamma-synchrony role-filler binding), arguing against scale-solves-reasoning on comparative grounds.
- **Core knowledge = the start-up software (Spelke & Kinzler 2007)** — four (+1) innate, domain-specific systems (objects/agents/number/geometry/social), each = individuation principles + a signature limit (~4 set-size, Weber ratio). This is the **extreme-prior end of slow-W**: structure that must exist *before* content binding, not learned from token statistics — and the origin of ARC's Core Knowledge Priors and the missing objectness prior (Gap #9). Uniquely human cognition is *combining* these mutually-encapsulated priors (a binding problem under a serial-attention bottleneck), not acquiring a new one. Object priors *contradict* agent priors (agents act at a distance), so the composition engine — natural language (Spelke) vs. an innate LoT (Revencu & Csibra) — is unresolved (the developmental face of Gaps #2 and #3).
- **Recursion = the combinatorial engine (Hauser, Chomsky & Fitch 2002)** — FLB (shared perceptual/semantic modules) vs. FLN (recursion alone, uniquely human). Recursion is the *productivity* facet where transformers/MLC hit the error ceiling (SCAN-length); the domain-general-recursion hypothesis (navigation → number → language) is the evolutionary form of the wiki's "reuse grid/graph machinery for abstract reasoning" claim.
- **Cultural learning = the acquisition channel (Tomasello & Rakoczy 2003)** — the leap is a *social acquisition channel*, not a bigger solitary reasoner: **imitation** (copy the demonstrator's latent *intention*/goal) vs. **emulation** (copy surface affordances) is the model-building/shortcut split at the acquisition layer. Vocabulary is *transmitted*, not individually reinvented — reframing Gap #3 partly as inverse-goal-inference from demonstration (the Phase-2 imitation route). Meta-representation (theory-of-mind) is *bootstrapped* from recursive sentential-complement syntax, not an innate module — a reflective level learnable from the right training distribution.
- **Neoteny = the developmental window (Sherwood 2008; Somel 2009)** — the extended plastic phase (human brain 27% of adult size at birth) *is* the long meta-learning outer loop; molecular neoteny is **mosaic (~4% of transcriptome), gray-matter/synaptic-specific, peaking at adolescent pruning** — i.e. a *selective deferral of slow-W consolidation*, not a uniform slowdown. Design reading: a **graded plasticity taper** that freezes structure *late*, plus domain-general→specific *emergence* (reweight attention/control/WM/inhibition under the schedule) rather than hand-built per-skill modules.

### The Learning Algorithm Lineage

```
Hopfield (1982)
  ↓ stochastic → Boltzmann Machine (1985) — learns P(data); intractable Z
      ↓ CHL (Contrastive Hebbian Learning) approximation of BM gradient
Contrastive Hebbian Learning (Movellan 1990)
  ↓ full clamp (β→∞) → mode-mismatch bug (J_CHL can be negative)
Equilibrium Propagation (Scellier & Bengio 2017)
  ↓ weak nudge (β small) → exact gradient via Theorem 1; no backward circuit
      [Parallel track]
Predictive Coding / FEP
  ↓ local error propagation; replaces Z with tractable F = −ELBO
TEM (2020) / TEM-t (2022)
  ↓ factorized ELBO; cross-environment structural transfer; emergent grid/place cells
```

**Biologically plausible learning (Gap 4):** Bartunov et al. 2018 ([[wiki/papers/bartunov-scalability-bio-dl-2018.md]]) quantified the scaling gap — FA, DTP, SDTP (biologically plausible optimization algorithms) all reach >93% top-1 error on ImageNet vs. backprop's 71%. EqProp excluded (computationally prohibitive at scale). The exactness-vs-symmetry tradeoff between EqProp (exact, requires W_{ij}=W_{ji}) and TP (biased approximation, no symmetry constraint) is the unresolved practical design choice. **Differentiable plasticity** (Miconi/Schmidgall) adds a third candidate: outer-loop BPTT discovers local Hebbian + neuromodulated update rules; deployed system adapts using only local synaptic rules; the retroactive neuromodulation variant (eligibility trace + delayed DA-like signal) is the most biologically plausible. Analog hardware that physically relaxes to equilibrium is the expected substrate for EqProp at scale.

### Architecture Determines Feasibility

Neural manifold constraints ([[wiki/concepts/neural-manifolds.md]]) make this an architectural necessity, not an efficiency preference:

- Standard transformer: abstract relational generalization **outside its intrinsic manifold** — unreachable regardless of training or scale
- TEM's g/x factorization: relational generalization **inside its reachable manifold by construction**
- LLMs/LRMs: knowledge-bounded — in-context fast loop cannot generalize beyond pretraining distribution to genuinely novel graph structures (McClelland 1995 theorem instantiation)

**Shortcut reasoning as the measurement problem:** Beger et al. 2025 show that standard accuracy metrics conflate shortcut solutions (correct output, wrong reasoning) with genuine concept acquisition. AI models produce correct-unintended rules ~27–29% of the time on ARC (humans ~5%), exposing that benchmark accuracy inflates apparent abstraction. **Missing objectness prior:** the most systematic ARC failure — models treat grids as pixel matrices, not scenes of discrete bounded entities. Token-prediction training never acquires Spelke's Core Knowledge objectness (persistence, coherent movement, boundaries); this is architectural, not a data-scale problem. See [[wiki/concepts/shortcut-reasoning.md]].

### The W/M Split Spectrum

| Model | Structural basis (W) | Task-specific code (M) | Timescale |
|---|---|---|---|
| Reservoir computing | Random, frozen | Linear readout, one-shot | W never trained; LSM Theorem 1: universal fading-memory with separation property |
| TEM | Slow backprop across environments | Hebbian, per-environment | W: slow; M: fast |
| Standard DNN | Jointly trained (entangled) | None (no factorization) | Single timescale |
| Meta-RL LSTM (Wang 2018) | Outer-loop weights = slow W | Hidden state = fast M policy | W: meta-training; M: within-episode |
| **VSA/HRR (Joffe 2025)** | None — pre-given DSL + circular convolution | SSP path integration per task | No slow W; isolates vocabulary co-discovery as the residual gap |
| **mPFC schemas** | Schema graph = slow W; organized by vmPFC over days | HC fast-M for specific episode instances | W: consolidation over days; M: single-trial BTSP |

**Reservoir → TEM gap (Gap 5):** LSM establishes universal fading-memory as the starting condition; TEM W is the endpoint (structured, transferable basis). How in-liquid plasticity adapts toward natural stimulus distributions, and how structural W emerges from a random basis without destroying universality, is open.

### The Extended Architecture: 11 Functional Blocks + GNW Hub + DMN Buffer

The full model decomposes into four regions plus a broadcasting hub (see [[wiki/queries/building-blocks-mec-hc-pfc.md]]):

**MEC (structural engine):** 1A Grid substrate (toroidal g) → 1B Path integration (continuous SO(N) rotation) → 1C Landmark correction (Kalman update)

**HC (fast binding):** 2A Bidirectional binding (g↔x) → 2B Pattern completion (iterative modern Hopfield) → 2C Importance gate (SWR/prediction-error write) → 2D Sparse allocation (top-k engram). **Interneuron cooperative control (Valero 2025):** four GABAergic families (PV/SST/VIP/Id2) have orthogonal computational roles in CA1 spatial coding — PV gates spatial computation speed; SST suppresses non-local inputs; VIP provides top-down disinhibitory release; Id2 (NDNF) controls long-range EC input via SLM. VIP→SST disinhibition enables mPFC top-down signals to open the HC gate. HC-mPFC frequency dissociation (Sigurdsson 2016): **gamma (encoding/sample phase)** via direct vHPC→mPFC monosynaptic channel; **theta (retrieval/choice phase)** via indirect Nucleus Reuniens relay.

**PFC (hierarchy and goals):** 3A Transformation Inferrer — **absent from base TEM** (inverse path integrator; set-attention over Δg → W posterior) → 3B Working memory + planning (TRNN episodic + meta-RL LSTM policy; **STA** as the planning mechanism — T neural subspaces encode the full future trajectory simultaneously; world model A_{ij} embedded in inter-subspace connections; attractor dynamics infer optimal plan in parallel; conveyor belt execution requires no replanning) → 3C Hierarchical stack — **parallel multi-level policy search** (all frontal levels active simultaneously from trial 1: pmPFC/tactic = 1st-order [Matsuzaka 2012], prePMd = 2nd-order, DLPFC = 3rd-order, FPC/BA-10 = 4th-order; reward-gated depth pruning; putamen→cortex→caudate Granger-causality chain [Badre 2010]; pmPFC integrates tactic + spatial cue + action while downstream pre-SMA receives only abstracted tactic + action [Awan 2020]) → 3D Goal generator (g_goal → argmin W). **vmPFC→MEC→NGF schema gate:** vmPFC signals control dCA1 ensemble overlap, determining whether new memories are assimilated into existing schema engrams (co-allocation) or kept separate (accommodation). **mPFC→RE→HC relay:** Nucleus Reuniens converts unidirectional HC→mPFC into a closed bidirectional loop; mPFC goal representations reach HC via mPFC→RE→sHPC for goal-directed forward planning; RE modulates plasticity in both targets simultaneously. **PFC-DNC molecular gate:** NE-α2A-AR→Gi→cAMP↓→HCN closure → PFC pyramidal network strengthened (Gap #8 NA-analog gate resolved at ion-channel level); catecholamine surge → cAMP flood → HCN/KCNQ open → network collapses (stress → reflexive behavior; molecular implementation of the integration→segregation switch).

**GNW Hub (broadcasting):** Long-range L2/3+L5 neurons select one representation from competing processor outputs and ignite global broadcasting via NMDA recurrence (~300ms onset). This provides the workspace-update mechanism between PFC (Prefrontal Cortex) reasoning cycles — but **offset ignition is absent** (Ferrante 2025), meaning the release/replacement mechanism is empirically unspecified. PFC (Prefrontal Cortex) encodes abstract category (not identity/orientation), confirming the abstract-hub/perceptual-satellite architecture: the hub manages relational tokens, not sensory pixels. **Binding interface (Awan 2020):** the hub does not query the satellite — it suppresses the satellite's direct sensory access, absorbs raw+abstract inputs internally, then releases an abstracted signal for satellite execution (suppress-then-release pattern).

**DMN (contextual buffer):** Integrates episodic memory, semantics, self-reference, and language into **frames of thought** — coherent context patterns that persist across event boundaries and transition at narrative/goal shifts. **Autopilot mode:** small local prediction errors → DMN maintains global generative model with minimal perturbation; the ongoing context frame persists. **Global model update mode:** large prediction error (scene change, goal shift, social model revision) → DMN recruits endogenous priors to reconstruct the generative model; high DMN engagement across heterogeneous domains. **Unique balanced output:** the DMN distributes effective connectivity output approximately equally across all cortical types (koniocortical through agranular) — the only network in the brain that does this; makes DMN the universal context broadcast module. **Two-tier internal structure:** receiver periphery (inferior parietal/precuneus, low-E1) collects convergent sensory input and desynchronizes during external tasks (→ faster RT); insulated core (anterior cingulate/mPFC, high-E1, agranular) generates abstracted internal content and remains suppressed longest during external tasks (→ higher accuracy). Task-general backbone: DMN within-network connectivity (C2C component 6) is the single most stable element of the whole-brain functional connectome across 7 cognitive states (Yoo 2022). **SN gating:** anterior insula causally suppresses DMN via high-frequency gamma — the macro-scale attention gate between internal and external processing modes. See [[wiki/entities/default-mode-network.md]].

**IWMT synthesis:** Safron 2022 proposes that the hub should implement turbo-coding — inter-level loopy belief propagation that iteratively refines predictions across cortical hierarchy levels. SOHMs (Spherical Organizing Hierarchical Maps) broadcast via the rich-club network, achieving integrated spatiotemporal-causal world modeling. This is a theoretical synthesis of IIT, GNWT, and FEP-AI; empirical validation is open. See [[wiki/entities/iwmt.md]].

**Hardware layer:** Each region implements the **canonical microcircuit** — L4 receives afferent input (thalamic or long-range cortical), seeds recurrent amplification via the L4→L2/3→L5→L6→L4 loop, and selects hypotheses via horizontal WTA (Winner-Take-All) inhibition. Feedforward projections (L3→L4, gamma, high SLN%) carry prediction errors; feedback projections (L5/6→L1/5/6, beta, low SLN%) carry predictions. The laminar anatomy enforces the PC (Predictive Coding) message hierarchy structurally.

### A Concrete Proposed Model (ARC-AGI-3 target)

[[wiki/queries/proposed-reasoning-model-block-architectures.md]] turns the block inventory into ranked, buildable architecture menus: **two factorized subsystems joined by a continuous↔discrete codec**, driven by the coupled loop and one neuromodulatory bus. Most blocks are off-the-shelf or minor adaptations; the whole reduces to **~5 inventions**, of which two carry almost all the risk. Highest-leverage picks: **1B path integration** = SSP fractional binding (analytic inverse → free reverse edges for 3A) with a learned SO(N) generator escape hatch for non-abelian rules; **3A Transformation Inferrer** = Neural-Process latent set encoder (calibrated diffuse-vs-sharp posterior gates exploration); **memory core** = Vector-HaSH (fuses 1A+2B+2D, no catastrophic forgetting); **D2 schema slow-W** = conflict-*triggered* accommodation store [INVENT #2]; **D3 planner** = STA (exploit, once `W` learned) ⇄ replay-MCTS (explore, structure latent); **3D goal engine** = value head + blended intrinsic reward (LP+InfoGain+empowerment) [INVENT #1, the verifier-free crux]. The gate is a VQ-in / mask+FiLM-out codec bound by mixed-selectivity conjunctive tuning in vmPFC. **The two inventions that carry the risk:** the intrinsic-motivation engine as internal verifier (3D) and the conflict-triggered schema accommodation trigger (D2) — structural generalization without forgetting.

### Convergent Evolution: Strongest Empirical Argument

At least 5–6 independent evolutionary lineages ([[wiki/concepts/convergent-allocentric-coding.md]]) derive the same **expansion → compression** circuit for allocentric world modeling:

| System | Expansion | Compression | Est. age |
|---|---|---|---|
| Vertebrate HC | Dentate gyrus | CA3 autoassociation | ~500 Mya |
| Insect Central Complex | Ellipsoid body ring | Fan-shape body | >400 Mya |
| Arthropod mushroom body | Kenyon cells | MBONs | >500 Mya |
| Cephalopod vertical lobe | Amacrine cells | Large-field cells | ~500 Mya |
| Crustacean hemiellipsoid | Kenyon-like | Projection neurons | ~500 Mya |
| Polychaete mushroom body | Kenyon-like | Lobus superior | ~600 Mya |

Conclusion: this motif is not just a good design choice — **convergent evolution argues it is near-optimal** for the computational problem of allocentric world modeling. The insect CX (Central Complex) is the cleanest ML implementation target (full Drosophila connectome known; ring attractor confirmed in vivo by Seelig & Jayaraman 2015). HC adds cross-environment W generalization that no other system has demonstrated.

## Key Open Problems

1. **Type 2 task gap (transformation inference)** — TEM and all current factorized models require the action vocabulary to be given externally. Discovering latent edge labels from (observation_before, observation_after) pairs — as required by ARC-AGI — is architecturally absent.
2. **Multi-level meta-graph** — **Biological architecture partially resolved (Badre et al. 2010):** parallel multi-level policy search — all frontal levels active simultaneously; reward-gated depth pruning; step-wise sigmoid learning curves as empirical signature. **Hierarchical STA stacking (Jensen et al. 2026):** stacking two STAs at different abstraction levels produces plans exponentially long in hierarchy depth; the abstract STA computes a coarse plan and the concrete STA fills in fine-grained steps. **Remaining gap:** learning appropriate abstractions to define the abstract state space for the higher-level STA is unsolved — the formal algorithm for how abstractions are discovered and how the resulting hierarchy is trained remains open.
3. **Vocabulary co-discovery** — the action alphabet itself must be inferred alongside graph structure. LAPA (VLA survey 2025) demonstrates vocabulary co-discovery via VQ-VAE on frame differences, but is domain-specific. No model handles vocabulary co-discovery with cross-environment meta-graph generalization simultaneously.
4. **Biologically plausible slow W** — EqProp (Scellier & Bengio 2017) resolves the theoretical problem: local contrastive Hebbian at equilibrium computes exact gradients. The practical problem remains: symmetric weights required; free-phase depth scaling is exponential on digital hardware; no large-scale empirical validation. Targetprop is the alternative (no symmetry required, but high-bias; Bartunov 2018 shows both fail at ImageNet). Differentiable plasticity (Gap candidate c) makes the deployed rule biologically plausible but still requires BPTT meta-training or evolutionary outer loop. Analog neuromorphic hardware is the expected resolution path for EqProp.
5. **Reservoir → structured basis** — LSM Theorem 1 establishes universal fading-memory as the starting condition; TEM W is the endpoint (structured, transferable). The developmental trajectory — how in-liquid plasticity tunes separation property toward natural statistics, and how W emerges without destroying universality — is unspecified.
6. **Active inference** — FEP (Free Energy Principle) extended to minimize expected future free energy (epistemic foraging) connects to goal-directed reasoning; formalism resolved (G(π) = epistemic + pragmatic value; Taniguchi 2023); architectural integration into the 11-block model unresolved. Mode-2 (CEM, non-probabilistic) vs. active inference (probabilistic posterior over z) design choice open.
7. **Workspace release/update mechanism** — Ferrante 2025 shows PFC (Prefrontal Cortex) offset ignition is absent at content transitions; no current architecture specifies how the GNW (Global Neuronal Workspace) hub releases one representation and admits the next. Must design an explicit workspace-update trigger.
8. **Abstract hub / perceptual satellite split** — Ferrante 2025 shows PFC (Prefrontal Cortex) encodes category but not identity or orientation; a reasoning model needs two distinct tiers: (a) a hub module operating on discrete/abstract representations + (b) a posterior module maintaining fine-grained perceptual content. **Progress:** anatomical grounding via DMN receiver periphery / insulated core split (Paquola 2025); dynamic gating via LC-NA integration toggle (Shine 2016); molecular gate via PFC-DNC NE-α2A-AR→cAMP→HCN cascade (Arnsten 2010); binding interface via suppress-then-release pattern (Awan 2020). **Remaining gap:** how the insulated core dynamically selects which receiver-periphery representations to query; network-level implementation of the suppress-then-release pattern is unresolved.
9. **Objectness prior** — Beger et al. 2025 show frontier AI systematically lacks Spelke's Core Knowledge objectness prior on ARC (objects persist, move coherently, have boundaries). Token-prediction training does not produce objectness representations; ~27–29% of AI "correct" ARC solutions use pixel/connectivity shortcuts (vs. ~5% humans). The architectural mechanism for acquiring discrete-entity priors without hand-coding maps to Blocks 1A+1B (structural scaffold) and remains open.
10. **Schema consolidation mechanism** — schemas require mPFC to generate NREM SWA during sleep to drive HC→neocortex transfer (Mander 2013); a reasoning model's consolidation cannot be passive. How schema conflict triggers controlled schema modification (accommodation) without disrupting unaffected associations is unresolved at the circuit level (the vmPFC→MEC circuit controls assimilation vs. accommodation but not the cellular mechanism of selective modification).
11. **Non-stationary topology (new 6th hardness source)** — the instance-graph edge set rewrites mid-episode; no wiki architecture re-infers topology online within the fast-M window. Learnable iff the rewrite is itself a stationary rewrite-graph; the lifting construction (rule-state as a node dimension) restores stationarity but needs a *factorized rule-code* to stay tractable, and requires a genuine third timescale only for Nomic-class self-amendment. Open sub-questions: (a) does STA wall-gating generalize from edge *masking* to edge *appearance*, or is appearance a distinct continual-structure-learning problem? (b) how deep must the timescale recursion go? (c) is frontier failure on Baba Is AI — where all four tractability axes are at their easy pole — a *planning-over-self-edits* deficit or a *compositional-generalization* deficit? A separate human-facing cost (Fluxx/Batman Fluxx: 10+ simultaneous active rules) is a working-memory tracking load distinct from the in-principle unsolvability pole.
12. **The composition engine (combining encapsulated core priors)** — uniquely human cognition = *composing* mutually-encapsulated, attention-competing core-knowledge priors (object priors literally contradict agent priors), not acquiring a new prior. The engine is unresolved: natural language (Spelke) fails the bidirectional-link test (core concepts are rarely spoken; function words have no core counterpart), while an innate LoT (Revencu & Csibra) faces the combinatorial-explosion double bind and a core-output-format codec problem. This is the developmental-psychology face of Gap #2 (continuous↔discrete codec) and Gap #3 (composition/vocabulary selection). See [[wiki/concepts/core-knowledge.md]], [[wiki/concepts/recursion.md]].
13. **Memorization/generalization routing** — no architecture (a) detects, for a new observation, whether it falls in a pure-memorization regime (Nanda's micro/macrofeature criterion: no input-general regularity predicts output beyond entity identity → distributed lookup-table storage is the *only* option, generalization provably impossible) vs. a generalizing regime, or (b) routes memorization-regime content to an isolated fast-M store while routing generalizable content through slow-W structural learning. Standard transformers compile everything into the same slow weights — a candidate root of both catastrophic-forgetting susceptibility and the subtle-memorization shortcut. See [[wiki/papers/fact-finding-factual-recall-nanda-2023.md]].
14. **The System-1→System-2 graft** — relational reinterpretation specifies *what* System 2 must add (four PSS properties) but not *how* it discovers role-filler structure from non-relational perceptual input without hand-coded predicates, nor how the higher-order system queries and reinterprets the lower-order store (Penn's "that it does, not how" — Bermúdez's redescription critique). LISA hand-codes the predicates; the autonomous-relational-learning and graft-interface problems are open, and the physical-level hominin change (PFC synchrony? long-range connectivity?) is unspecified. See [[wiki/concepts/relational-reinterpretation.md]].
15. **Goal bootstrapping / intrinsic verifier (ARC-AGI-3 crux)** — the verifier-free, latent-goal regime has no external learning signal, so the goal must be bootstrapped from intrinsic reward. Open: does LP+empowerment survive as a Phase-3 verifier or does self-generated reward reintroduce reward-hacking? How is the *region partition* whose learning-progress is tracked bootstrapped before `W` (the latent-graph partition) exists (chicken-egg)? And what is the cheap neural estimator for empowerment `I(A;S')`? See [[wiki/queries/reasoning-as-coupled-navigation-strategizing.md]].

## Promising Directions

- **Inverse path integration (Block 3A)** — architecturally: set-attention over Δg vectors to produce soft posterior over W vocabulary; W jointly trained by forward and inverse path integration
- **EqProp on analog hardware** — the theoretical barrier to biologically plausible slow-W is resolved (Theorem 1); the path forward is analog circuits that physically relax to equilibrium (eliminating the O(L³) digital free-phase cost). Memristor/phase-change crossbar arrays are the candidate substrate.
- **STA as Block 3B planning module** — Jensen et al. 2026: embed world model in inter-subspace connections; attractor inference of optimal trajectory avoids sequential rollout error accumulation; validated on dynamic-reward mazes; conveyor belt dynamics match El-Gaby et al. 2023 PFC recordings. Extending STA to abstract relational graphs (ARC-AGI rule spaces rather than spatial grids) is the key open question — requires verifying whether A_{ij} can encode symbolic transitions.
- **Memory schemas as mPFC slow-W** — schemas are organized mPFC knowledge graphs that implement the slow W of latent-graph storage. The vmPFC→MEC→NGF circuit biases which HC neurons co-encode new memories into existing schema engrams (assimilation) vs. fresh neurons (accommodation). An explicit schema-gating module that controls HC encoding (not just retrieval) is the direct architectural implication. Schema accommodation (conflict-triggered controlled update) is the biological analog of EWC with a conflict-detection trigger.
- **JEPA world models for planning** — V-JEPA 2-AC (Assran et al. 2026) validates representation-space Mode-2 planning at scale: CEM in latent space achieves zero-shot robot pick-and-place; temporally straight latent trajectories ([[wiki/concepts/world-models.md]]) simplify CEM search. Extension to abstract reasoning (ARC-AGI grid transformations) would require a JEPA trained on structured discrete transformations rather than video. Violation-of-expectation (VoE) probing detects whether the world model has internalized rules without labeled data.
- **DMN as contextual buffer** — a reasoning model needs a DMN-equivalent module: a slow-timescale context integrator that maintains narrative frames, is suppressed by external-attention demands (salience gate), and provides a universal broadcast to all processing levels. The insulated core (slow, agranular, abstracted) + receiver periphery (fast, laminated, sensory) two-tier structure maps onto Gap #8's abstract hub + perceptual satellite split, suggesting the DMN architecture itself as a design template for the contextual layer.
- **PFC-DNC as NA-analog gate** — the molecular gate for WM capacity and the integration/segregation switch is now specified: NE-α2A-AR→Gi→cAMP↓→HCN closure → network strengthened (Gap #8 NA-analog gate). A model that implements this as a per-module gain-control parameter (tuned by a global arousal signal) and a stress-collapse mechanism would implement the dynamic hub activation required by Gap #8. Guanfacine (α2A agonist) already demonstrates clinical efficacy for ADHD WM deficits.
- **Canonical microcircuit as architecture template** — the L4→L2/3→L5→L6→L4 loop is a solved recurrent-amplification + soft-WTA + two-output design. A reasoning model block that seeds a recurrent loop with a sparse goal signal and reads out hypothesis selection (L2/3 WTA) and action (L5) separately is more anatomically faithful than flat transformer layers.
- **TEM + CSCG unification** — fast de-novo mapping (CSCG) for novel environments + structural transfer (TEM) for familiar domains; would formalize the HC map → memory transition
- **TRNN for working memory (Block 3B)** — self-inhibition + sparse + hierarchical topology; outperforms attractor RNN on WM capacity, spatial navigation, distractor robustness; γ parameter approximates D1/D2 balance; directly applicable as the fast-M layer
- **Neuromodulation circuit as meta-learning** — DA/ACh/5-HT/NA jointly implement the outer metalearning loop; many-to-one interactions (Mei 2025) allow attractor landscape control and three-factor rules with NA as a surprise signal; D1/D2 cellular mechanisms now grounded at receptor level; connecting PVLV to the Block 3B WM gate is a tractable next step
- **SNN gamma-band binding** — LISA model demonstrates gamma-band temporal synchrony for role-filler binding at WM capacity ≤2–3 propositions; an SNN substrate at the binding stage (Block 2A/2B) would implement this with exponentially fewer units than a rate-coded equivalent (CD_n theorem)
- **Insect CX (Central Complex) as minimal implementation target** — ring attractor heading (1A/1B) + P-EN path integration (1B) + PFL goal vector (3D) fully instantiated in a known connectome; a CX-inspired module could provide the MEC core without requiring emergence
- **Dual-channel ARC evaluation** — Beger et al. 2025's accuracy + rule-quality evaluation exposes the accuracy/abstraction dissociation; implementing this for ARC-Prize scoring would force genuine concept acquisition rather than shortcut exploitation
- **Three-phase developmental curriculum** — explore (curiosity-driven world-model SSL) → imitate (behavioral cloning on cheap, manually-recorded human step-traces) → practice (verifier-gated RL on the agent's own rollouts). Imitation targets the single most expensive object in latent-graph discovery — the *path* — which is why a cheap dataset of human step-by-step solutions has outsized value; but it must be sandwiched between SSL pretraining (a vocabulary to imitate *in*) and verifier-gated practice (to survive off its own distribution and exceed the teacher). Directly maps AlphaGo's SFT→self-play to abstract reasoning.
- **Intrinsic-motivation goal engine** — replace the single DA-novelty term with a blended `r_int = w₁·LP + w₂·InfoGain + w₃·Empowerment − w₄·cost`; learning progress defeats the noisy-TV trap and auto-organizes a Goldilocks curriculum, empowerment keeps exploration in controllable regions. This is the candidate internal substitute for the missing external verifier in ARC-AGI-3 (Block 3D), and the design's biggest departure from the solver stack. *Ingest candidates to formalize: Oudeyer & Kaplan 2007 (LP/IAC), Schmidhuber (compression progress), Klyubin 2005 (empowerment), Gottlieb & Oudeyer 2018.*
- **Neoteny-inspired graded plasticity taper** — Somel 2009 localizes the human developmental delay to synaptic/gray-matter (slow-W) genes at the adolescent-pruning stage. Design analog: a *per-subsystem* plasticity schedule that freezes structural weights *late* rather than a uniform learning-rate decay, keeping `W`/schema malleable across all curriculum phases and consolidating only after practice — a principled pretraining→finetuning schedule.
- **SSP path integration + Neural-Process transformation inference** — SSP fractional binding gives an *analytic* operator inverse (free reverse edges for Block 3A) and grid periodicity emergent from a learnable frequency matrix; a Neural-Process latent set encoder over `{Δgᵢ}` returns a calibrated posterior over the operator whose diffuse-vs-sharp uncertainty natively gates exploration. Together they attack the two hardest blocks (1B, 3A) with mostly-existing components. See [[wiki/queries/proposed-reasoning-model-block-architectures.md]].
- **Conflict-triggered schema slow-W (accommodation trigger)** — the mPFC assimilation/accommodation mechanism as an EWC-style store gated by a *learned conflict detector* (schema-graph prediction error) rather than a fixed regularizer: cross-attention routes small targeted edits (assimilate) vs. fresh allocation (accommodate). The store is easy; the conflict-gated write *schedule* is where structural generalization without forgetting lives — a high-value invention target.

## Major Controversies

- **GNW offset ignition absent (Ferrante 2025)** — preregistered adversarial collaboration (n=256, fMRI+MEG+iEEG) confirms onset ignition but finds no PFC (Prefrontal Cortex) offset ignition at content transitions; GNW's symmetric onset/offset prediction fails the causal test; constitutive vs. enabling PFC (Prefrontal Cortex) role is the active resolution. Both GNW (Global Neuronal Workspace) and IIT (Integrated Information Theory) failed at least one preregistered prediction — no clean winner.
- **PFC constitutive vs. enabling** — iES null result (Raccah 2021, n=100 intracranial) and category/identity dissociation (Ferrante 2025) both challenge GNW's claim that PFC (Prefrontal Cortex) activity *constitutes* conscious experience; the working resolution is that PFC (Prefrontal Cortex) is necessary but not sufficient (enabling), with the posterior cortex as the constitutive locus. Architectural implication: the hub module manages selection, not the phenomenal content.
- **TBT evolutionary claim** — that every cortical column recapitulates the HC formation (MEC/LEC/HC as L6/L4/L2-3) is interpretive; significant circuit differences exist; efference copy driving L6 confirmed only in motor cortex
- **Manifold plasticity** — motor cortex manifold constraints shown hard (BCI reversal failure); HC manifold appears more plastic (UMAP drifts with learning); whether hard structural manifold boundaries exist throughout cortex is empirically open
- **Grid code universality** — abstract grid codes shown in vmPFC during 2D conceptual navigation (Constantinescu 2016); extension to non-2D domains (causal graphs, linguistic structure) is unconfirmed
- **EqProp exactness vs. practical viability** — Theorem 1 is exact in the limit β→0; finite β introduces approximation; symmetric weights required; no result beyond MNIST. The claim that EqProp is the resolved solution to Gap 4 is theoretical only — experimental validation at scale is absent.
- **CA3 connectivity and capacity** — Sammons 2023 found 9–11% pyramidal-to-pyramidal connectivity (10× above prior estimates); this significantly increases estimated biological capacity but is a single dataset
- **Mode-2 as System I** — Lett 2025 argues that LeCun's Mode-2 planning (MPC with hard-wired CEM) is deterministic System I behavior, not flexible System II; genuine System II requires learning the planning algorithm itself. V-JEPA 2-AC's success validates Mode-2 mechanically but leaves the System II gap open. Simultaneous plasticity of world model + cost + control algorithm requires a stabilizing meta-management architecture (currently unspecified).
- **Shortcut rate interpretation** — Beger et al. 2025 report ~27–29% shortcut rate on ConceptARC "correct" solutions; this measures rule quality, not task success. Whether this rate reflects an intrinsic limitation of discriminative training or a fixable data/evaluation issue is contested; the objectness prior analysis suggests it is architectural.
- **HC as cognitive map vs. universal relational memory** — Kumaran & Maguire 2005: HC activates uniquely for spatial/temporal relational graph traversal but NOT for topology-matched social relational graphs, challenging the view that HC is a general-purpose relational memory system. If confirmed broadly, this constrains TEM-like models to domains with metric or sequential embedding; amodal schema organization must rely on a different (mPFC-led) system.
- **PFC attention vs. WM maintenance** — Lebedev et al. 2004 (monkey dlPFC): 61% of spatially tuned neurons track the *attended* location vs. 16% tracking the *remembered* location during delay periods; regional dissociation (ventrolateral = attention-dominant, dorsomedial = mixed). This challenges the Goldman-Rakic view that PFC's delay-period firing is primarily WM maintenance; PFC may be doing dynamic attentional tracking rather than (or in addition to) static holding. The architectural implication: Block 3B should distinguish attentional pointer from stored content.
- **Solvers do research math vs. LGD is the unsolved bottleneck** — the 2026 math survey documents an LLM-generator + external-verifier stack producing IMO-2025 gold, MiniF2F 25%→93%, and Lean-verified Erdős solves, superficially refuting "latent structure inference is unsolved (<5% frontier)." Resolution (complementary, not contradictory): the solver stack *externalizes* the graph (mathlib/DSL), the verifier (Lean kernel), and the vocabulary (given libraries) — it never learns an *internal* latent graph, so it wins where infrastructure exists and ceilings exactly where vocabulary is latent (FrontierMath Tier-4 single-digit). The two literatures are opposite faces of one wall; the shared frontier is vocabulary co-discovery. See [[wiki/queries/brain-inspired-vs-solver-approach.md]].
- **Verified vs. self-reported ARC-AGI-2 frontier** — ARC-Prize-verified results are 24% (open Kaggle, cost-capped) and 54% (audited refinement harness), while closed labs self-report ≤85% (GPT-5.5) on unreproducible harnesses; the ~30-pt gap is unverifiable until models+data are open, and high closed-model scores may partly reflect knowledge coverage on the shared task distribution. Burden of proof is on the labs.
- **Core systems unitary vs. dissociable sub-principles** — Spelke treats core systems as all-or-none (an entity failing one object principle loses all objecthood), using this to argue the composite social-agent concept is *not* a core system. Revencu & Csibra counter (Stahl & Feigenson 2015): infants selectively explore by *which* principle was violated, and continuity dissociates from cohesion. Design reading: a core prior is a *bundle of separately-maskable sub-constraints*, not a monolithic binary gate — the objectness prior (Gap #9) should be dissociable sub-principles.
- **Sparse localist (biological SDR) vs. dense polysemantic superposition (LLM recall)** — biological engrams/SDRs achieve capacity via sparse (2–6%), near-orthogonal coding; Nanda 2023 shows Pythia 2.8B stores facts via *dense* polysemantic superposition (near-zero cross-fact neuron correlation), not localist engrams. May reflect a write-timescale constraint (sparsity required for one-shot Hebbian binding in biological HC, not for slow batch-consolidated memorization) rather than a true contradiction.
