# Core Knowledge — Spelke & Kinzler, Developmental Science 2007

**Citation:** Spelke, E. S. & Kinzler, K. D. (2007). Core knowledge. *Developmental Science* 10(1), 89–96. doi:10.1111/j.1467-7687.2007.00569.x

---

## Key Insights

- **Neither general-purpose nor massively modular.** The mind is built on a *small number* of core systems, not a single learning device (Locke/Hume/connectionism) nor thousands of special-purpose modules (Fodor's target). Flexible skills build on these foundations. This is the conceptual middle ground a reasoning architecture should target.
- **Four core systems, each with individuation principles + signature limits:**
  - **Objects** — cohesion/continuity/contact; ~3–4 set-size limit (parallel individuation); present in newborns and newly-hatched chicks; *no* core subcategories for food/artifact/liquid.
  - **Agents** — goal-directed, efficient (rational action), contingent, reciprocal, gaze-sensitive; explicitly *not* bound by object principles (agents act at a distance, on discontinuous paths). Mirror-neuron substrate.
  - **Number** — Approximate Number System: cross-modal, abstract, addable/subtractable; scalar variability → Weber ratio limit; precision improves ratio 2.0 (6mo) → 1.15 (adult).
  - **Geometry** — distance/angle/sense of extended surfaces; reorientation uses layout geometry and *ignores* non-geometric cues (color); landmark use is a dissociable associative process.
- **Signature limits are the cross-population diagnostic.** The same limits recur across human infants, non-human primates, and remote cultures with minimal counting/number language (Pirahã object tracking; Mundurukú ANS = educated French adults), establishing the systems as universal and evolutionarily ancient, not cultural.
- **Fifth candidate system: social partners.** Us/them coalition detection; language/accent (not race/gender) is the most reliable cue; in-group preference by 3–6 months.
- **Core knowledge both scaffolds and misleads.** Word learning, counting, maps, and cultural learning build on core concepts; but the same priors cause errors (non-Euclidean space, discontinuous quantum objects, us/them bias) — overcome only via effortful conceptual change.

---

## Limitations

- A synthesis/review, not a single experiment; the number-system principles remain debated (three competing formal accounts).
- The fifth (social) system is explicitly a candidate, not established.
- Says *what* the priors are, not *how* to instantiate them computationally — the open engineering problem this wiki inherits (core-knowledge concept, open problem #1).
