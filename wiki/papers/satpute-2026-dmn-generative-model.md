# The default mode network in a hierarchical generative model of the brain

Satpute, A.B. (2026). *Current Opinion in Behavioral Sciences*, 69:101656. https://doi.org/10.1016/j.cobeha.2026.101656

---

- DMN acts as an "orchestra conductor" distributing high-level predictions simultaneously across all cortical networks and laminar types, while receiving selective input from higher-order association areas only — contrasted with a "chain-of-command" sequential hierarchy
- Three neuroanatomical axes structure DMN organization: (1) longitudinal (posterior=sensory/external/alpha; anterior=internal/autonomic/theta); (2) local hierarchical (agranular→eulaminate III gradient within each complex); (3) layered communication (Barbas laminar-type rule: similar laminar profiles preferentially connect, enabling parallel rather than serial cross-complex exchange)
- Abstraction operates as dimensionality reduction: higher cortical hierarchy = fewer neurons + broader receptive fields + slower transmission (reduced myelination) + broader projections → compressed lower-dimensional priors
- DMN operates in two modes: **autopilot** (stable predictions → minimal generative model perturbation) and **global model update** (large prediction error from scene changes, goal shifts, affective transitions → wholesale reconstruction using endogenous priors)
- Heterarchical architecture: local hierarchies within the anterior and posterior complexes + lateral parallel exchange between complexes following the laminar-type rule; not purely top-down

**Limitations:** review paper without primary data; three-axis model is a proposal based on existing histology and connectivity data; layered communication axis is the hardest to test empirically (requires laminar-resolved task fMRI). Draws heavily on Paquola et al. 2025 cytoarchitectural data.
