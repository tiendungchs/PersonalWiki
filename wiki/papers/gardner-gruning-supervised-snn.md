# Supervised Learning in SNNs for Precise Temporal Encoding — Gardner & Grüning

**Citation:** Gardner B & Grüning A. Supervised Learning in Spiking Neural Networks for Precise Temporal Encoding. University of Surrey. (PLoS ONE, ~2016)

---

## Key Computational Insights

- **Filtered error achieves near-maximal temporal encoding capacity:** FILT rule (exponentially smoothed postsynaptic error, τ_q ≈ τ_m) achieves memory capacity α_m ≈ 0.14–0.15 patterns per synapse at 1 ms timing precision, matching the E-learning Chronotron and approaching the theoretical maximum (0.1–0.3, Memmesheimer et al. 2014). INST rule (instantaneous/point-process error) achieves only α_m ≈ 0.07.

- **Stability from phase portrait analysis:** FILT produces a stable fixed point at the desired weight w*; INST creates an unstable attractor causing persistent spike-timing fluctuations. Mechanism: exponential filtering converts the discontinuous ΔW jump at the firing threshold into a smooth approach, so the weight converges rather than oscillates. General principle: filtered/continuous error signals → stable attractors; point-process errors → unstable dynamics.

- **Sub-millisecond temporal precision:** FILT maintains α_m ≈ 0.07 at Δt = 0.2 ms; INST fails completely below 0.8 ms. Temporal codes are therefore quantitatively viable at timescales an order of magnitude finer than individual EPSP widths — consistent with the coherent convergence argument (Gerstner 1996).

- **PSP as theoretically grounded eligibility trace:** both rules derive the postsynaptic potential ε as the presynaptic learning factor from maximum-likelihood gradient ascent (Pfister et al. 2006), not heuristic STDP. FILT's weight update minimises a van Rossum Distance-like error between target and actual spike trains; the STDP-shaped learning window emerges from theory, not empirical measurement.

- **Supervised signal from referent activity templates:** biologically, target spike trains must arrive from "referent activity templates" generated by other circuits. Urbanczik & Senn 2014 provide a compartmental mechanism: somatic input nudges firing toward the target; dendritic synapses learn the input-pattern association. Structurally analogous to cerebellar climbing-fiber supervision — error injected at the soma/thick dendrite, weights updated at the distal contact (parallel fiber → Purkinje synapse).

## Limitations

- Tested only on small feedforward networks (200–600 presynaptic neurons); scaling to 10⁴ synapses per neuron (biological scale) not demonstrated.
- No recurrent connections; temporal code confined to isolated feedforward layers.
- Source of referent activity templates is asserted, not mechanistically derived — the biological plausibility gap shared by all supervised SNN rules.
