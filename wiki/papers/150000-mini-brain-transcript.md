# Thousand Brains Theory — 150,000 Mini-Brains (transcript)

**Source:** `raw/150000-mini-brain-transcript.txt` (Hawkins / Numenta video lecture)

---

## Key Computational Insights

- **Universal columnar architecture:** neocortex = ~150,000 cortical columns each performing identical computation using a miniaturized HC formation circuit: L4 = LEC (sensory "what"), L6 = MEC (grid-like path integration, "where"), L2-3 = HC (binding + lateral consensus voting).
- **Sensorimotor coupling is primitive:** each column builds its reference frame by pairing sensory observations (L4) with motor-derived location estimates (L5 efference copy → L6 path integrator). The "where" signal is *generated* by the column's own motor predictions, not received from a dedicated MEC.
- **Distributed inference without central integrator:** columns communicate only via L2-3 lateral connections; mutually consistent hypotheses amplify (rich-get-richer via E/I dynamics); the single percept emerges from consensus, not hierarchical readout.
- **Hierarchical abstraction = abstract path integration:** higher cortical columns receive lower-column L5 outputs as their sensory input; "motor commands" become attention shifts; conceptual-space navigation uses the same L6 grid-like mechanism as physical navigation.
- **Independent derivation of g/x/p factorization:** TBT (Thousand Brains Theory) arrives at structural/sensory/conjunctive split from evolutionary anatomy rather than formal modeling — cortex evolved by transplanting the HC formation circuit at scale.

## Limitations

- No computational model at the scale of 150,000 columns with full consensus voting; convergence unproven.
- Consensus voting mechanism (lateral L2-3 convergence, inhibition dynamics) is mechanistically motivated but not formally specified.
- Evolutionary HC→column claim is interpretive; direct developmental evidence incomplete.
