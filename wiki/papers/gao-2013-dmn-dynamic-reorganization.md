# The Dynamic Reorganization of the Default-Mode Network During a Visual Classification Task

**Citation:** Gao W, Gilmore JH, Alcauter S, Lin W (2013). Frontiers in Systems Neuroscience, 7:34. DOI: 10.3389/fnsys.2013.00034

---

- 4-state seed-based fcMRI (PCC seed) + group ICA across 19 adults: pre-task rest (R1) → relaxed visual task (T1) → intense visual task (T2) → post-task rest (R2); hierarchical letter-classification task with congruent/incongruent trials and overlapping scenic pictures
- DMN decomposes into **three functionally distinct subsets** during task engagement: (1) stable core (PCC, mPFC, bilateral IPL, LTC — connectivity unchanged across all 4 states), (2) decreasing receiver periphery (precuneus, bilateral angular gyrus, cerebellar vermis — progressively desynchronize with task demand), (3) increasing salience integration (bilateral insula/IFC, ACC, MCC — not connected at rest; progressively couple with stable core as demand increases)
- **Dissociated behavioral correlates:** within-network desynchronization (stable↔decreasing) correlates *only* with faster RT (R²=0.33–0.36, p<0.01); outside-network integration (stable↔increasing) correlates *only* with higher accuracy (R²=0.24–0.35, p<0.03); no cross-correlation observed
- Both patterns fully reversible post-task (R2 returns to R1 baseline; no significant R1–R2 difference detected in any connectivity measure)
- Results confirmed by two independent methods (seed-based PCC analysis and group ICA), spatial correlation between methods r=0.50–0.71

**Limitations:** N=19 (3 excluded from R2); single visual classification task — generalization to other task types uncertain; SNR confound cannot be fully excluded despite bidirectional connectivity changes within the same stable region set; no trial structure, preventing event-related analysis.
