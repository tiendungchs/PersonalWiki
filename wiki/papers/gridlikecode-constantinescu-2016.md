# Organizing conceptual knowledge in humans with a gridlike code

Constantinescu, O'Reilly & Behrens. *Science* 352(6292), 1464–1468. 2016. DOI: 10.1126/science.aaf0941

---

## Key Computational Insights

- **Grid code generalizes from spatial to abstract 2D conceptual dimensions.** Humans navigating a learned "bird space" (neck × leg length) showed hexagonal (6-fold symmetric) fMRI signal modulation — cos(6θ) regressor — in vmPFC and entorhinal cortex, identical to the spatial navigation signature. Subjects had no conscious awareness of a spatial structure, confirming the code is implicit.
- **Grid angle is stable over 30 min and >1 week.** The hexagonal orientation estimated in one session predicted BOLD modulation in a held-out session, both within-day (t₂₆=2.61, p<0.05) and cross-day (t₂₀=3.65, p<0.01). vmPFC and ERH grid angles aligned with each other (t₂₁=2.18, p<0.05), suggesting a shared coordinate frame across regions.
- **Grid strength predicts task performance.** Hexagonal modulation in vmPFC correlated with accuracy (r=0.43, p=0.04); grid-angle consistency correlated independently with performance (r=0.43, p=0.04). More gridlike representation = better conceptual generalization, linking the neural code to behavioral output.
- **The conceptual grid network spans vmPFC, ERH, OFC, PCC, RSC, LPC, and TPJ.** This is both the spatial navigation network and the default mode network — anatomically broader than the classical hippocampal-entorhinal boundary. Suggests grid-based relational coding is a cortex-wide organizing principle.
- **Only 6-fold symmetry is present; 4-, 5-, 7-, and 8-fold controls are all null.** The hexagonal specificity rules out generic fMRI confounds and confirms the signal reflects the same mechanism as spatial grid cells, not generic modulation.

---

## Limitations

- fMRI measures bulk population BOLD, not single grid cells; hexagonal symmetry is inferred from phase-aligned signal modulation, not direct recording.
- Only tested 2D continuous abstract spaces; applicability to discrete, high-dimensional, or non-Euclidean relational structures is untested.
- The fMRI quadrature test used to define ROIs can overestimate Z-scores; statistical inference rests on the cross-validation procedure, not the whole-brain map.
