# 20 Years of the Default Mode Network: A Review and Synthesis

**Menon, V. (2024).** _Nature Reviews Neuroscience_ (or equivalent comprehensive review).

---

## Key Computational Insights

- **Integrate-and-broadcast model:** DMN integrates self-reference, social cognition, episodic memory, and language/semantic memory, then broadcasts unified representations as "frames of thought" — spatiotemporal patterns that transition at event boundaries, constituting the internal narrative.
- **Triple-network switching:** Salience network (anterior insula, dACC) causally suppresses DMN when external demands arise (confirmed via intracranial EEG high-frequency gamma 76–200 Hz and rodent optogenetics), enabling frontoparietal engagement; all three networks are anti-correlated.
- **PCC as causal hub:** Dynamic causal modeling and information-theoretic analysis of intracranial EEG reveal net causal outflow *from* PCC and mPFC to sensory/motor networks — not the reverse; hub status is causal, not merely correlational.
- **Slow timescale:** DMN integrates over longer temporal receptive windows than any other brain network (shown via naturalistic speech processing); neural patterns in PCC persist across event boundaries, resetting only at perceived context transitions.
- **HC-DMN coupling for episodic integration:** Individual DMN nodes track distinct episodic features — hippocampus tracks encoding/retrieval success; cortical nodes (RSC, mPFC) track value-, emotion-, and detail-laden features; the system integrates new information "for further deliberation, constrained by past experiences."

---

## Limitations

- fMRI temporal resolution (~1s) is too coarse to resolve DMN frame dynamics; intracranial EEG is spatially sparse because core DMN nodes (PCC, RSC, vmPFC) are not seizure foci.
- Core DMN functions (inner speech, mind wandering, semantic self-narrative) are uniquely human with no adequate animal model, limiting mechanistic translation.
