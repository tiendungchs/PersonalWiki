# A default mode of brain function

**Raichle et al., PNAS (2001)**

The foundational paper that first proposed a default mode of brain function. Used PET-measured oxygen extraction fraction (OEF) as a proxy for the resting metabolic baseline; observed that certain regions consistently *deactivate* during goal-directed tasks rather than activate.

**Key insights:**
- OEF is approximately uniform at rest → evidence for a true default state (not noise)
- Task-induced deactivations are task-independent: the same regions suppress across diverse task types, implying a stable resting baseline
- Proposed functional roles: PCC/precuneus = continuous environmental monitoring; MPFC = salience evaluation of monitored information

**Limitations:** PET only (~30s temporal resolution); no connectivity data; functional interpretations superseded by fMRI-era reviews (see Menon 2024).
