# Inferences on a Multidimensional Social Hierarchy Use a Grid-like Code

**Citation:** Park SA, Miller DS, Boorman ED (2021). *Nat. Neurosci.* 24, 1292–1301. https://doi.org/10.1038/s41593-021-00916-3

The active-decision, value-coupled complement to Garvert (passive/implicit) and Qu (developmental) — grid-like coding of *inferred* trajectories through a **discretely-sampled** 2D social hierarchy, and the first Axis-1 source to tie the grid code directly to **decision-value construction**. Axis-1 #2 of the central-framing audit.

## Key computational insights

- **Paradigm:** n=21. 16 faces ("entrepreneurs") ranked in two *independent* social dimensions (competence, popularity), each learned on a **separate day** via feedback-based binary comparisons of **adjacent (1-rank-difference) pairs only**. Subjects were never shown the 1D or 2D structure — the 4×4 hierarchy must be reconstructed by transitive inference *within* a dimension and integration *across* dimensions. The fMRI "partner selection" task requires choosing the better collaborator by the pair's **growth potential (GP)** (area set by the higher rank in each dimension) — a *novel* 2D inference never trained and given no feedback.
- **Piecemeal cross-day observations → one unitary 2D map in HC+EC.** RSA: Mahalanobis pattern dissimilarity in a-priori HC and EC ROIs scales with true **2D Euclidean distance**, significantly stronger than either 1D rank distance alone; MDS recovers the layout; whole-brain searchlight adds mPFC/OFC and PCC/precuneus/RSC.
- **Grid-like (6-fold hexagonal) modulation for *inferred* direct trajectories** (F0→F1, F0→F2 vectors never traversed or trained) in **EC and mPFC** (aligned>misaligned; specific to 6-fold vs. null 4/5/7/8-fold), plus **TPJ and STS** (theory-of-mind / perspective-taking regions, engaged because subjects choose *on behalf of* F0). Grid orientation is consistent across sessions **>1 week apart**, and the effect is robust for **first-ever-seen pairs** → trajectories composed on the fly, not recall of past decisions.
- **The grid code gates decision value — the paper's novel contribution.** GP (the decision value) is encoded in **mPFC, TPJ, and EC**; GP coding is **significantly stronger for grid-aligned than misaligned trajectories** (6-fold specific), and subjects with stronger EC grid modulation show stronger GP coding in TPJ (r=0.44–0.58). The value-comparison term |GP1−GP2| appears in vmPFC + EC/HC. → the grid geometry is *used to construct the values that drive choice*, not merely present during the task.
- **Map built without task demand.** Two separate 1D number lines would pass every criterion; subjects nonetheless spontaneously integrate them into a 2D map with a hexagonal code — the map/grid is the brain's *default* solution even absent an explicit 2D demand (a biological natural-attractor datapoint).

## Limitations

- Training **engineered equidistant ranks** and imposed a high-accuracy threshold to make the hypothesized grid measurable — the metric 2D geometry is partly a design artifact (Chen 2022's "contrived-task" caveat), and excluded low-accuracy participants may have used other geometries.
- The space is **2D-Euclidean-embeddable and orderable** (4×4 ranks) → validates the *metric/embeddable* slice, not non-embeddable symbolic structure. BOLD-only (no single-unit); grid↔value coupling is correlational, not causal perturbation.
