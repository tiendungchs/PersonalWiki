## Architectural Gaps

Read at the start of every session. Update when new gaps.

Capabilities the target reasoning model needs but that no current page fully addresses.

| # | Gap | Block | Status |
|---|-----|-------|--------|
| 1 | **Transformation inference (Block 3A)** — no current architecture infers latent edge labels from (observation_before, observation_after) pairs; required for ARC-AGI Type 2 tasks. **Partial engineering instance (Das et al. 2025, [[wiki/papers/das-compositional-neurosymbolic-arc-2025.md]]):** proposes edge labels as compositions over a *fixed 22-primitive DSL* and infers the shared label by **cross-example consistency intersection** (`Π* = ∩_i {π : π ⊨_i}`, min-depth tie-break) — a concrete Block-3A mechanism reaching 24.4% ARC-AGI-2. Limitation: the edge alphabet is hand-supplied (see Gap #3), and consistency is approximated by repeated LLM pattern detection rather than exact symbolic search. | 3A | Open — concrete pipeline exists but with pre-given primitive alphabet |
| 2 | **Multi-level meta-graph** — flat W cannot represent compositional rule chains; Block 3C three-level PFC (Prefrontal Cortex) hierarchy needs formalization as a nested latent graph. **Biological architecture resolved (Badre et al. 2010):** The frontal hierarchy (PMd/1st-order → prePMd/2nd-order → mid-DLPFC/3rd-order → FPC/4th-order) implements *parallel* multi-level policy search — all abstraction levels are active simultaneously from the first observation of a new learning episode; no sequential handoff. prePMd activates for 2nd-order rule search from trial 1, even before any 2nd-order rule is known; activity is reward-gated (declines when no higher-order rule is available). When a higher-order rule is discovered, all covered lower-order rules become immediately learnable (step-wise sigmoid learning curve). Fronto-striatal direction: putamen gates cortical rule updates; cortex drives caudate value learning — a directed putamen→cortex→caudate chain. **Hierarchical STA stacking (Jensen et al. 2026):** stacking two Spacetime Attractors at different abstraction levels (abstract STA → concrete STA) produces plans exponentially long in hierarchy depth vs. linear scaling for a flat STA; the abstract STA computes a coarse plan and the concrete STA fills in fine-grained steps toward each abstract sub-goal. This provides the computational mechanism for parallel hierarchical planning. **Remaining gap:** learning appropriate abstractions to define the abstract state space for the higher-level STA remains unsolved; the formal algorithmic model for how abstractions are discovered and the resulting hierarchy is trained remains open. **Two-format refinement (mec-abstract query; [[wiki/queries/building-blocks-declarative-subsystem.md]]):** the meta-graph is not one nested structure but **two structurally different representational formats that must gate each other** — a continuous/metric grid scaffold (System 1: MEC→HC→vmPFC/DMN, cheap phase-shift matching) and a discrete/declarative rule+schema system (System 2: PFC rostro-caudal hierarchy + mPFC schema slow-W, RL search). The gating interface is *not* a missing module: it is two already-named bidirectional channels — ventral-HC→mPFC (metric context selects which schema) inbound, and vmPFC→MEC→NGF / mPFC→RE→HC (schema masks metric allocation & traversable edges) outbound — whose residual difficulty is a **continuous↔discrete codec** at each end (quantize `p=f(g,x)`→schema key; schema→inhibitory mask over SO(N) edges / engram writes). A single-substrate model fails on whichever format it lacks. | 3C | Biological architecture resolved; hierarchical STA provides partial computational solution; abstraction-learning + two-format codec unsolved |
| 3 | **Vocabulary co-discovery** — the action alphabet must be inferred alongside graph structure; no current model handles both simultaneously without a pre-given symbol set. **Partial solution:** LAPA (VLA survey 2025) learns a discrete action codebook jointly with a visual world model via VQ-VAE on frame differences — demonstrates vocabulary co-discovery is tractable; limitation is domain-specificity (manipulation video only, no cross-environment meta-graph generalization). **(brainstorm) Social-acquisition route (Tomasello & Rakoczy 2003; [[wiki/concepts/cultural-learning.md]]):** human vocabulary is *not* individually reinvented — it is transmitted through a cultural channel where imitation copies the demonstrator's *intention* (the latent symbol/goal), not surface behavior. Suggests vocabulary co-discovery may be far more tractable via *demonstration/distillation* (the construal is supplied by another agent) than via pure unsupervised inference — reframing Gap #3 partly as an inverse-goal-inference problem rather than a solitary structure-learning one. **(brainstorm) Program-level route — library learning (Goodman et al. 2024; DreamCoder/LAPS; [[wiki/concepts/probabilistic-language-of-thought.md]]):** in the PLoT framing the "vocabulary" is the primitive set of a program-generating meta-program, and it is co-discovered *by abstracting shared sub-programs out of solved tasks* — learned concepts are added as primitives, evolving the effective language of thought (shorter programs → changed inductive bias). DreamCoder demonstrates this at scale; LAPS shows natural-language jointly bootstraps the library. This makes the action alphabet an emergent product of compression over past solutions rather than a pre-given or purely-unsupervised set — the program-space analog of LAPA's action codebook, and the mechanism the social/demonstration route (above) would supply construals *into*. Limitation: DreamCoder's library search is itself intractable (PLoT's inference wall). **Hand-supplied-vocabulary baseline (Das et al. 2025, [[wiki/papers/das-compositional-neurosymbolic-arc-2025.md]]):** the CoreThink ARC reasoner uses a *fixed, human-curated* 22-pattern DSL (no co-discovery) and still reaches SOTA-at-release 24.4/30.8% — a useful control showing how much a good pre-given alphabet buys, and an upper-bound target for what co-discovery must eventually match; its own limitation section flags the fixed DSL as incomplete and the main ceiling, re-motivating co-discovery. **Biological route — contextual inference ([[wiki/queries/action-semantics-contextual-inference.md]]):** the brain never learns "what action A does"; it maintains a *repertoire* of context-conditioned action→outcome models and infers a posterior over which applies (COIN, Heald/Lengyel/Wolpert 2021; mechanistic ancestor MOSAIC, Haruno/Wolpert/Kawato 2001, where each module's *responsibility* = softmax over its own forward-model prediction error). Arbitrary-visuomotor-mapping work (Wise & Murray) dissociates the circuits: HC + ventral PFC acquire *novel* arbitrary mappings, premotor–striatal circuits retain *overlearned* ones. Design consequence: the action index is a **pointer**, its effect `v` a **latent variable inferred at play-time** from `(obs_t, obs_{t+1})` — never a trained `action → effect` function (which averages conflicting contexts). **Sharp limit:** BCI learning (Sadtler et al. 2014; Golub et al. 2018) shows fast re-binding is *reassociation within the pre-existing intrinsic manifold*; outside-manifold mappings take days. Contextual inference therefore buys vocabulary **selection**, not **invention** — it closes the "new environment, known operator alphabet" case and leaves the hard core of this gap (an operator never previously seen) fully open. | 3A | Partial (LAPA); social/demonstration + program-level library-learning routes identified, not integrated; fixed-DSL baseline quantifies the target; biological route (contextual inference) resolves *selection* but not *invention* |
| 4 | **Biologically plausible slow W** — PC (Predictive Coding) local updates (`Δw ∝ ε·x`) support within-network learning; cross-environment structural W-generalization via local rules remains open. Two main candidates: **(a) EqProp (Scellier & Bengio 2017)** — theoretically exact (Theorem 1) via two-phase energy minimization; ΔW ∝ (1/β)[ρ(u^β)ρ(u^β) − ρ(u⁰)ρ(u⁰)] computes ∂J/∂W exactly as β→0; but requires symmetric weights W_{ij}=W_{ji} and has not been scaled beyond MNIST. **(b) Targetprop (Bengio et al. 2015)** — high-bias approximation via DAE theorem: gradient ≈ [f(g(h))−h]/σ²; no weight symmetry required (f and g trained independently); frames learning as variational EM (Expectation Maximization) (E-step = inference dynamics, M-step = local weight update); demonstrated on MNIST, collapses at ImageNet. **Bartunov et al. 2018 quantify the gap**: FA/DTP/SDTP all >93% top-1 error on ImageNet vs. BP's 71%; EqProp excluded (computationally prohibitive). Target diversity (low-entropy output alphabet) is a concrete TP bottleneck. The exactness-vs-symmetry tradeoff between EqProp and targetprop is the unresolved design choice. **(c) Differentiable plasticity (Miconi et al. 2018)** — outer-loop BPTT through plasticity parameters η discovers local Hebbian + neuromodulated update rules; *deployed* system adapts using only local synaptic rules with fixed η; addresses slow-W by making the algorithm biologically plausible at deployment time, even though meta-training still requires BPTT (or evolutionary search as a biologically plausible outer loop). Retroactive neuromodulation variant (eligibility trace e_ij + delayed dopamine-like M) is the most biologically plausible: only local signals at deployment. Limitation: O(N²T) memory during BPTT meta-training; poor generalization when plasticity search space is unconstrained. See [[wiki/concepts/differentiable-plasticity.md]]. | 2A–2D | Open |
| 5 | **Reservoir → structured basis** — **LSM Theorem 1 (Maass et al. 2002) establishes the starting point**: a generic random liquid with SP already achieves universal fading-memory computation. TEM W is the endpoint: a structured basis transferable across environments. The gap is two-fold: (a) how in-liquid plasticity (explicitly proposed by Maass et al. as necessary for developmental SP tuning) adapts SP toward natural stimulus distributions, and (b) how structural W gradually emerges from a random reservoir through learning while preserving the universality guarantee. | 1A | Open |
| 6 | **Active inference** — expected free energy / epistemic foraging (FEP extended) connects to goal-directed reasoning. **Formalism resolved (Taniguchi et al. 2023):** G(π) = epistemic value (information gain, exploration) + pragmatic value (preference-matching, exploitation). Active inference vs. CaI (entropy-regularized RL) distinction now explicit. **Remaining gap:** which module in the 11-block architecture computes G(π) and gates configurator/action selection; CaI vs. active inference design choice unresolved. LeCun's Mode-2 (CEM, non-probabilistic) vs. active inference (probabilistic posterior over z) comparison now concretized by V-JEPA 2-AC | 3D | Formalism resolved; integration open |
| 7 | **Workspace release/update mechanism** — Ferrante 2025 shows PFC (Prefrontal Cortex) offset ignition is absent at content transitions; no current architecture specifies how the global broadcasting hub releases one representation and admits the next; GNW-style hub design assumes symmetric onset/offset, but the evidence only supports onset; must design an explicit workspace-update trigger | Hub module | Open |
| 8 | **Abstract hub / perceptual satellite split** — Ferrante 2025 shows PFC (Prefrontal Cortex) encodes category but not identity or orientation; a reasoning model therefore needs two distinct tiers: (a) a hub module operating on discrete/abstract representations + (b) a posterior module maintaining fine-grained perceptual content; the binding interface between these tiers (how the hub selects and queries the satellite) is unspecified. **Anatomical evidence (Paquola et al. 2025):** the DMN's insulated core (anterior cingulate, medial PFC; high-E1, agranular) and receiver periphery (inferior parietal, precuneus; low-E1, eulaminate) directly instantiate these two tiers anatomically within one distributed network. The binding mechanism (how the insulated core queries the receiver periphery) remains the unresolved design question. **Dynamic evidence (Shine et al. 2016):** frontoparietal, striatal, and thalamic hub nodes mediate whole-brain integration by elevating between-module participation coefficient B_T during demanding tasks (N-back); this integration is neuromodulatorily gated (LC-NA: pupil r=0.241 with B_T) and scales with task complexity. The hub module must therefore be *conditionally activated* by a global arousal/demand signal, not always-on. Design implication: hub layer + global gain gate (NA analog) implements the integration/segregation switch; static hub connectivity is insufficient. **Molecular grounding (Arnsten et al. 2010):** the NA analog gate has a known molecular implementation — NE-α2A-AR → Gi → cAMP ↓ → HCN channels close → PFC pyramidal network strengthened; overdriven catecholamines → cAMP surge → HCN/KCNQ open → network collapses. This specifies the gate as a cAMP-modulated ion-channel shunt operating on seconds timescales. See [[wiki/concepts/pfc-dynamic-network-connectivity.md]]. | Hub module | Open — binding interface unresolved; dynamic gating mechanism molecularly grounded |
| 8 (cont.) | **Behavioral evidence for two-tier split (Gao et al. 2013):** Task-based fMRI decomposes DMN into stable core (PCC, mPFC, IPL, LTC), receiver periphery (precuneus, ANG — desynchronizes during tasks), and salience integration (insula, ACC, MCC — couples during tasks). Dissociated behavioral correlates: receiver-periphery desynchronization → faster RT; salience integration → higher accuracy. This independently confirms the anatomical two-tier split from task-functional data — and reveals that the two tiers serve separable cognitive roles (speed vs. accuracy). The binding interface question remains: how does the stable core gate salience-network recruitment selectively based on task demands? | Hub module | Open — binding interface unresolved |
| 8 (cont.) | **Biological implementation of hub-satellite binding via spatial gating (Awan et al. 2020):** In multi-tactic primate tasks, pmPFC (abstract hub) integrates tactics + spatial cue + action; pre-SMA (satellite) receives tactics + action but *not* spatial cue; SMA receives action only. The binding interface mechanism: pmPFC processes raw sensory info (cue location) internally and passes only the abstracted tactic+action signal to pre-SMA, which has abundant PFC afferents but sparse direct parietal input. The hub does **not** query the satellite for sensory details — it absorbs those details itself, then releases a filtered abstract signal for satellite execution. Schizophrenia evidence (PFC hypoactivity → compensatory pre-SMA overactivation) confirms that the hub exerts top-down *suppression* of the satellite's sensory channel when engaged. **Design implication:** the binding interface operates as suppress-then-release: hub engages → satellite's direct sensory access is suppressed → hub integrates raw+abstract → hub disengages → satellite executes with abstracted signal only. This concretizes the "how does the hub select and query the satellite" question in Gap #8: the hub does not query; it filters and gatekeeps. See [[wiki/papers/neuronal-tactic-sensorimotor-awan-2020.md]] and [[wiki/entities/prefrontal-cortex.md]]. | Hub module | Binding interface partially grounded — suppress-then-release pattern established at single-area scale; network-level implementation open |
| 9 | **Objectness prior** — Beger et al. 2025 show that frontier AI models on ConceptARC systematically lack Spelke's Core Knowledge objectness prior (objects persist, move coherently, have boundaries), defaulting to pixel/color/connectivity shortcuts instead. Token-prediction training does not produce objectness representations; the architectural mechanism for encoding discrete-entity priors without hand-coding is unspecified. Objectness is one of four innate core-knowledge systems (objects/agents/number/geometry; [[wiki/concepts/core-knowledge.md]], [[wiki/papers/spelke-kinzler-core-knowledge-2007.md]]) — this gap generalizes beyond objectness to *all four* start-up priors. Maps to Blocks 1A+1B (structural scaffold). See [[wiki/concepts/shortcut-reasoning.md]]. **Quantified from the fix side (Das et al. 2025, [[wiki/papers/das-compositional-neurosymbolic-arc-2025.md]]):** hand-coding the objectness parse (deterministic 8-connected components + bounding box / centroid / canonical shape / cavity features) is the +6.9pp dominant driver on ARC-AGI-2 (vs. +3.9pp self-consistency), directly measuring how much of the gap *is* the missing discrete-entity parse. But it is objectness *supplied*, not *acquired* — the architectural mechanism for learning it remains unspecified, and a fixed connected-component parse inherits the 4/8-connectivity shortcut when true grouping is non-connectivity-based. | 1A | Open — value quantified; learned-acquisition mechanism still missing |
| 10 | **Memorization/generalization boundary — no mechanism routes content to a fast-M-like store vs. slow-W structural learning based on whether it is generalizable.** Nanda et al. 2023 ([[wiki/papers/fact-finding-factual-recall-nanda-2023.md]]) formalize the criterion via micro/macrofeatures: a task is pure memorization exactly when no macrofeature (input-general, task-relevant regularity) predicts the output beyond raw entity identity — in that regime distributed/superposed lookup-table storage is not an interpretability failure but the mathematically only option, and generalization is provably impossible. **Missing piece:** no architecture in this wiki (a) detects, for a new observation, whether it falls in a pure-memorization regime or a rules-of-thumb/generalizing regime, or (b) routes memorization-regime content to an isolated fast-M-like store while routing generalizable content through slow-W structural learning. Standard transformers do neither — all content is compiled into the same slow weights, a candidate mechanistic root for both catastrophic-forgetting susceptibility under fine-tuning and the "subtle memorization" shortcut failure mode ([[wiki/concepts/shortcut-reasoning.md]], MATH-Perturb 2025: applying a memorized technique without checking structural applicability is arguably the generalizing-task analog of using a memorized-fact lookup outside its valid domain). | 2A–2D / W-M split | Open — formal criterion identified; regime-detection and routing mechanism unspecified |
