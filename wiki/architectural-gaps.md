| 1 | **Transformation inference (Block 3A)** ‚Äî no current architecture infers latent edge labels from (observation_before, observation_after) pairs; required for ARC-AGI Type 2 tasks. **Partial engineering instance (Das et al. 2025, [[wiki/papers/das-compositional-neurosymbolic-arc-2025.md]]):** proposes edge labels as compositions over a *fixed 22-primitive DSL* and infers the shared label by **cross-example consistency intersection** (`ŒÝ* = ‚à©_i {œÄ : œÄ ‚ä®_i}`, min-depth tie-break) ‚Äî a concrete Block-3A mechanism reaching 24.4% ARC-AGI-2. | 2 | **Multi-level meta-graph** ‚Äî flat W cannot represent compositional rule chains; Block 3C three-level PFC (Prefrontal Cortex) hierarchy needs formalization as a nested latent graph. **Biological architecture resolved (Badre et al. 2010):** The frontal hierarchy (PMd/1st-order ‚Üí prePMd/2nd-order ‚Üí mid-DLPFC/3rd-order ‚Üí FPC/4th-order) implements *parallel* multi-level policy search ‚Äî all abstraction levels are active simultaneously from the first observation of a new learning episode; no sequential handoff. prePMd activates for 2nd-order rule search from trial 1, even before any 2nd-order rule is known; activity is reward-gated (declines when no higher-order rule is available). When a higher-order rule is discovered, all covered lower-order rules become immediately learnable (step-wise sigmoid learning curve). Fronto-striatal direction: putamen gates cortical rule updates; cortex drives caudate value learning ‚Äî a directed putamen‚Üícortex‚Üícaudate chain. **Hierarchical STA stacking (Jensen et al. 2026):** stacking two Spacetime Attractors at different abstraction levels (abstract STA ‚Üí concrete STA) produces plans exponentially long in hierarchy depth vs. linear scaling for a flat STA; the abstract STA computes a coarse plan and the concrete STA fills in fine-grained steps toward each abstract sub-goal. This provides the computational mechanism for parallel hierarchical planning. **Remaining gap:** learning appropriate abstractions to define the abstract state space for the higher-level STA remains unsolved; the formal algorithmic model for how abstractions are discovered and the resulting hierarchy is trained remains open. **Two-format refinement (mec-abstract query; [[wiki/queries/building-blocks-declarative-subsystem.md]]):** the meta-graph is not one nested structure but **two structurally different representational formats that must gate each other** ‚Äî a continuous/metric grid scaffold (System 1: MEC‚ÜíHC‚ÜívmPFC/DMN, cheap phase-shift matching) and a discrete/declarative rule+schema system (System 2: PFC rostro-caudal hierarchy + mPFC schema slow-W, RL search). The gating interface is *not* a missing module: it is two already-named bidirectional channels ‚Äî ventral-HC‚ÜímPFC (metric context selects which schema) inbound, and vmPFC‚ÜíMEC‚ÜíNGF / mPFC‚ÜíRE‚ÜíHC (schema masks metric allocation & traversable edges) outbound ‚Äî whose residual difficulty is a **continuous‚Üîdiscrete codec** at each end (quantize `p=f(g,x)`‚Üíschema key; schema‚Üíinhibitory mask over SO(N) edges / engram writes). A single-substrate model fails on whichever format it lacks. | 3C | Biological architecture resolved; hierarchical STA provides partial computational solution; abstraction-learning + two-format codec unsolved |
| 3 | **Vocabulary co-discovery** ‚Äî the action alphabet must be inferred alongside graph structure; no current model handles both simultaneously without a pre-given symbol set. **Partial solution:** LAPA (VLA survey 2025) learns a discrete action codebook jointly with a visual world model via VQ-VAE on frame differences ‚Äî demonstrates vocabulary co-discovery is tractable; limitation is domain-specificity (manipulation video only, no cross-environment meta-graph generalization). **(brainstorm) Social-acquisition route (Tomasello & Rakoczy 2003; [[wiki/concepts/cultural-learning.md]]):** human vocabulary is *not* individually reinvented ‚Äî it is transmitted through a cultural channel where imitation copies the demonstrator's *intention* (the latent symbol/goal), not surface behavior. Suggests vocabulary co-discovery may be far more tractable via *demonstration/distillation* (the construal is supplied by another agent) than via pure unsupervised inference ‚Äî reframing Gap #3 partly as an inverse-goal-inference problem rather than a solitary structure-learning one. **(brainstorm) Program-level route ‚Äî library learning (Goodman et al. 2024; DreamCoder/LAPS; [[wiki/concepts/probabilistic-language-of-thought.md]]):** in the PLoT framing the "vocabulary" is the primitive set of a program-generating meta-program, and it is co-discovered *by abstracting shared sub-programs out of solved tasks* ‚Äî learned concepts are added as primitives, evolving the effective language of thought (shorter programs ‚Üí changed inductive bias). DreamCoder demonstrates this at scale; LAPS shows natural-language jointly bootstraps the library. This makes the action alphabet an emergent product of compression over past solutions rather than a pre-given or purely-unsupervised set ‚Äî the program-space analog of LAPA's action codebook, and the mechanism the social/demonstration route (above) would supply construals *into*. Limitation: DreamCoder's library search is itself intractable (PLoT's inference wall). **Hand-supplied-vocabulary baseline (Das et al. 2025, [[wiki/papers/das-compositional-neurosymbolic-arc-2025.md]]):** the CoreThink ARC reasoner uses a *fixed, human-curated* 22-pattern DSL (no co-discovery) and still reaches SOTA-at-release 24.4/30.8% ‚Äî a useful control showing how much a good pre-given alphabet buys, and an upper-bound target for what co-discovery must eventually match; its own limitation section flags the fixed DSL as incomplete and the main ceiling, re-motivating co-discovery. | 3A | Partial (LAPA); social/demonstration + program-level library-learning routes identified, not integrated; fixed-DSL baseline quantifies the target |

| 5 | **Reservoir ‚Üí structured basis** ‚Äî **LSM Theorem 1 (Maass et al. 2002) establishes the starting point**: a generic random liquid with SP already achieves universal fading-memory computation. The gap is two-fold: (a) how in-liquid plasticity (explicitly proposed by Maass et al. as necessary for developmental SP tuning) adapts SP toward natural stimulus distributions, and (b) how structural W gradually emerges from a random reservoir through learning while preserving the universality guarantee. | 1A | Open |
| 6 | **Active inference** ‚Äî expected free energy / epistemic foraging (FEP extended) connects to goal-directed reasoning. **Formalism resolved (Taniguchi et al. 2023):** G(œÄ) = epistemic value (information gain, exploration) + pragmatic value (preference-matching, exploitation). | 7 | **Workspace release/update mechanism** ‚Äî Ferrante 2025 shows PFC (Prefrontal Cortex) offset ignition is absent at content transitions; no current architecture specifies how the global broadcasting hub releases one representation and admits the next; GNW-style hub design assumes symmetric onset/offset, but the evidence only supports onset; must design an explicit workspace-update trigger | Hub module | Open |
| 8 | **Abstract hub / perceptual satellite split** ‚Äî Ferrante 2025 shows PFC (Prefrontal Cortex) encodes category but not identity or orientation; a reasoning model therefore needs two distinct tiers: (a) a hub module operating on discrete/abstract representations + (b) a posterior module maintaining fine-grained perceptual content; the binding interface between these tiers (how the hub selects and queries the satellite) is unspecified. **Anatomical evidence (Paquola et al. 2025):** the DMN's insulated core (anterior cingulate, medial PFC; high-E1, agranular) and receiver periphery (inferior parietal, precuneus; low-E1, eulaminate) directly instantiate these two tiers anatomically within one distributed network. The binding mechanism (how the insulated core queries the receiver periphery) remains the unresolved design question. **Dynamic evidence (Shine et al. 2016):** frontoparietal, striatal, and thalamic hub nodes mediate whole-brain integration by elevating between-module participation coefficient B_T during demanding tasks (N-back); this integration is neuromodulatorily gated (LC-NA: pupil r=0.241 with B_T) and scales with task complexity. The hub module must therefore be *conditionally activated* by a global arousal/demand signal, not always-on. Design implication: hub layer + global gain gate (NA analog) implements the integration/segregation switch; static hub connectivity is insufficient. **Molecular grounding (Arnsten et al. 2010):** the NA analog gate has a known molecular implementation ‚Äî NE-Œ±2A-AR ‚Üí Gi ‚Üí cAMP ‚Üì ‚Üí HCN channels close ‚Üí PFC pyramidal network strengthened; overdriven catecholamines ‚Üí cAMP surge ‚Üí HCN/KCNQ open ‚Üí network collapses. This specifies the gate as a cAMP-modulated ion-channel shunt operating on seconds timescales. See [[wiki/concepts/pfc-dynamic-network-connectivity.md]]. | Hub module | Open ‚Äî binding interface unresolved; dynamic gating mechanism molecularly grounded |
| 8 (cont.) | **Behavioral evidence for two-tier split (Gao et al. 2013):** Task-based fMRI decomposes DMN into stable core (PCC, mPFC, IPL, LTC), receiver periphery (precuneus, ANG ‚Äî desynchronizes during tasks), and salience integration (insula, ACC, MCC ‚Äî couples during tasks). Dissociated behavioral correlates: receiver-periphery desynchronization ‚Üí faster RT; salience integration ‚Üí higher accuracy. This independently confirms the anatomical two-tier split from task-functional data ‚Äî and reveals that the two tiers serve separable cognitive roles (speed vs. accuracy). The binding interface question remains: how does the stable core gate salience-network recruitment selectively based on task demands? | Hub module | Open ‚Äî binding interface unresolved |
| 8 (cont.) | **Biological implementation of hub-satellite binding via spatial gating (Awan et al. 2020):** In multi-tactic primate tasks, pmPFC (abstract hub) integrates tactics + spatial cue + action; pre-SMA (satellite) receives tactics + action but *not* spatial cue; SMA receives action only. The binding interface mechanism: pmPFC processes raw sensory info (cue location) internally and passes only the abstracted tactic+action signal to pre-SMA, which has abundant PFC afferents but sparse direct parietal input. The hub does **not** query the satellite for sensory details ‚Äî it absorbs those details itself, then releases a filtered abstract signal for satellite execution. Schizophrenia evidence (PFC hypoactivity ‚Üí compensatory pre-SMA overactivation) confirms that the hub exerts top-down *suppression* of the satellite's sensory channel when engaged. **Design implication:** the binding interface operates as suppress-then-release: hub engages ‚Üí satellite's direct sensory access is suppressed ‚Üí hub integrates raw+abstract ‚Üí hub disengages ‚Üí satellite executes with abstracted signal only. This concretizes the "how does the hub select and query the satellite" question in Gap #8: the hub does not query; it filters and gatekeeps. See [[wiki/papers/neuronal-tactic-sensorimotor-awan-2020.md]] and [[wiki/entities/prefrontal-cortex.md]]. | Hub module | Binding interface partially grounded ‚Äî suppress-then-release pattern established at single-area scale; network-level implementation open |
| 9 | **Objectness prior** ‚Äî Beger et al. 2025 show that frontier AI models on ConceptARC systematically lack Spelke's Core Knowledge objectness prior (objects persist, move coherently, have boundaries), defaulting to pixel/color/connectivity shortcuts instead. Token-prediction training does not produce objectness representations; the architectural mechanism for encoding discrete-entity priors without hand-coding is unspecified. Objectness is one of four innate core-knowledge systems (objects/agents/number/geometry; [[wiki/concepts/core-knowledge.md]], [[wiki/papers/spelke-kinzler-core-knowledge-2007.md]]) ‚Äî this gap generalizes beyond objectness to *all four* start-up priors. Maps to Blocks 1A+1B (structural scaffold). See [[wiki/concepts/shortcut-reasoning.md]]. **Quantified from the fix side (Das et al. 2025, [[wiki/papers/das-compositional-neurosymbolic-arc-2025.md]]):** hand-coding the objectness parse (deterministic 8-connected components + bounding box / centroid / canonical shape / cavity features) is the +6.9pp dominant driver on ARC-AGI-2 (vs. +3.9pp self-consistency), directly measuring how much of the gap *is* the missing discrete-entity parse. But it is objectness *supplied*, not *acquired* ‚Äî the architectural mechanism for learning it remains unspecified, and a fixed connected-component parse inherits the 4/8-connectivity shortcut when true grouping is non-connectivity-based. | 1A | Open ‚Äî value quantified; learned-acquisition mechanism still missing |
| 10 | **Memorization/generalization boundary ‚Äî no mechanism routes content to a fast-M-like store vs. slow-W structural learning based on whether it is generalizable.** Nanda et al. 2023 ([[wiki/papers/fact-finding-factual-recall-nanda-2023.md]]) formalize the criterion via micro/macrofeatures: a task is pure memorization exactly when no macrofeature (input-general, task-relevant regularity) predicts the output beyond raw entity identity ‚Äî in that regime distributed/superposed lookup-table storage is not an interpretability failure but the mathematically only option, and generalization is provably impossible. **Missing piece:** no architecture in this wiki (a) detects, for a new observation, whether it falls in a pure-memorization regime or a rules-of-thumb/generalizing regime, or (b) routes memorization-regime content to an isolated fast-M-like store while routing generalizable content through slow-W structural learning. Standard transformers do neither ‚Äî all content is compiled into the same slow weights, a candidate mechanistic root for both catastrophic-forgetting susceptibility under fine-tuning and the "subtle memorization" shortcut failure mode ([[wiki/concepts/shortcut-reasoning.md]], MATH-Perturb 2025: applying a memorized technique without checking structural applicability is arguably the generalizing-task analog of using a memorized-fact lookup outside its valid domain). | 2A‚Äì2D / W-M split | Open ‚Äî formal criterion identified; regime-detection and routing mechanism unspecified |
